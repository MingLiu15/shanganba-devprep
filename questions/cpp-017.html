STL容器的分类和特点

C/C++ 中等
标签：STL, 容器, vector, list, map, set

📑 题目

介绍STL主要容器及其适用场景。

💡 参考答案

顺序容器：
- vector：动态数组，随机访问O(1)，尾部插入O(1)，中间插入慢
- deque：双端队列，首尾插入O(1)
- list：双向链表，插入删除O(1)，无随机访问

关联容器：
- set/map：红黑树，有序，O(log n)
- unordered_set/map：哈希表，无序，平均O(1)

容器适配器：
- stack、queue、priority_queue（基于其他容器实现）

选择原则：
- 频繁随机访问→vector
- 频繁插入删除→list
- 需要查找→unordered_map（快）或map（有序）

💻 代码示例

vector<int> vec = {1, 2, 3}; vec.push_back(4); map<string, int> scores; scores["Alice"] = 90; unordered_map<int, string> hash; hash[1001] = "Employee1"; set<int> s = {3, 1, 4, 1, 5}; // {1, 3, 4, 5}
