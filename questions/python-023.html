Python排序算法（Timsort）

Python 中等
标签：数据结构, 排序, Timsort, 算法

📑 题目

解释Python内置排序算法Timsort的原理及sorted()和list.sort()的使用。

💡 参考答案

Python使用Timsort排序算法，由Tim Peters于2002年实现，是一种混合稳定的排序算法。

Timsort原理：
- 结合了归并排序和插入排序的优点。
- 将数组划分为已排序的连续子序列（称为run）。
- 对较短的run使用插入排序（对小数组效率高）。
- 对已排序的run使用归并排序合并。
- 利用数据中已有的有序性（自然有序的run直接使用）。

时间复杂度：
- 最好情况（已排序）：O(n)。
- 平均情况：O(n log n)。
- 最坏情况：O(n log n)。
- 空间复杂度：O(n)。

sorted() vs list.sort()：
- sorted(iterable)：返回新列表，适用于任何可迭代对象。
- list.sort()：原地排序，返回None，只适用于列表。
- 两者都支持key参数（指定排序键）和reverse参数（降序）。

key参数的使用：
- key=str.lower：不区分大小写排序。
- key=lambda x: x[1]：按元组第二个元素排序。
- key=operator.itemgetter(1)：同上但更快。
- key=len：按长度排序。

稳定排序：相等元素的相对顺序不变，可以实现多级排序。

💻 代码示例

# sorted() 返回新列表 nums = [3, 1, 4, 1, 5, 9, 2, 6] print(sorted(nums)) # [1, 1, 2, 3, 4, 5, 6, 9] print(sorted(nums, reverse=True)) # [9, 6, 5, 4, 3, 2, 1, 1] print(nums) # 原列表不变 # list.sort() 原地排序 nums.sort() print(nums) # 已排序 # key参数 students = [('Tom', 85), ('Jane', 92), ('Bob', 85)] print(sorted(students, key=lambda x: x[1])) # 按分数 print(sorted(students, key=lambda x: x[0])) # 按姓名 # 多级排序（利用稳定性） students.sort(key=lambda x: x[0]) # 先按姓名 students.sort(key=lambda x: x[1]) # 再按分数 # 同分数的保持姓名排序 # 自定义对象排序 class Person: def __init__(self, name, age): self.name, self.age = name, age def __repr__(self): return f'{self.name}:{self.age}' people = [Person('Tom',25), Person('Jane',23)] print(sorted(people, key=lambda p: p.age))
