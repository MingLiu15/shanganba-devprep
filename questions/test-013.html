测试用例设计方法：等价类划分

自动化测试 简单
标签：测试设计, 等价类划分, 黑盒测试, 测试用例

📑 题目

什么是等价类划分法？如何应用？

💡 参考答案

**等价类划分定义：**
将输入数据划分为若干等价类，每个等价类中的数据对揭露程序错误具有等效作用。只需从每个等价类中选取一个代表进行测试。

**等价类类型：**
1. **有效等价类**：合理的、有意义的输入数据集合
2. **无效等价类**：不合理的、无意义的输入数据集合

**划分原则：**
- 如果输入条件规定了取值范围，则划分出一个有效等价类和两个无效等价类
- 如果输入条件规定了必须如何，则划分出一个有效等价类和一个无效等价类
- 如果输入条件是布尔量，则划分出一个有效等价类和一个无效等价类

**示例：用户名输入（6-20位字母数字）**
- 有效等价类：6-20位字母数字
- 无效等价类：<6位、>20位、含特殊字符、空值

💻 代码示例

# 等价类划分测试示例 import pytest def validate_username(username): """验证用户名：6-20位字母数字""" if not username: raise ValueError("用户名不能为空") if not isinstance(username, str): raise TypeError("用户名必须是字符串") if len(username) < 6: raise ValueError("用户名长度不能小于6位") if len(username) > 20: raise ValueError("用户名长度不能超过20位") if not username.isalnum(): raise ValueError("用户名只能包含字母和数字") return True # 测试用例 class TestUsernameValidation: # 有效等价类测试 def test_valid_min_length(self): assert validate_username("abc123") # 刚好6位 def test_valid_max_length(self): assert validate_username("a" * 20) # 刚好20位 def test_valid_normal(self): assert validate_username("testuser123") # 正常长度 # 无效等价类测试 def test_invalid_too_short(self): with pytest.raises(ValueError, match="小于6位"): validate_username("abc") # <6位 def test_invalid_too_long(self): with pytest.raises(ValueError, match="超过20位"): validate_username("a" * 21) # >20位 def test_invalid_special_char(self): with pytest.raises(ValueError, match="字母和数字"): validate_username("test@123") # 含特殊字符 def test_invalid_empty(self): with pytest.raises(ValueError, match="不能为空"): validate_username("") # 空值 def test_invalid_none(self): with pytest.raises(ValueError): validate_username(None) # None值
