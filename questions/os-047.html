TCP/IP协议栈详解

操作系统 中等
标签：TCP/IP, 网络协议, OSI, socket, 网络编程

📑 题目

TCP/IP协议栈的分层结构是什么？各层的主要协议有哪些？

💡 参考答案

**TCP/IP四层模型：**

1. **应用层：**
- HTTP/HTTPS：Web通信
- FTP：文件传输
- SMTP/POP3/IMAP：邮件
- DNS：域名解析
- SSH/Telnet：远程登录

2. **传输层：**
- TCP：面向连接，可靠传输
- UDP：无连接，快速传输
- 端口：标识应用进程

3. **网络层：**
- IP：逻辑寻址和路由
- ICMP：网络诊断（ping）
- ARP：IP地址转MAC地址

4. **链路层：**
- Ethernet：局域网通信
- MAC地址：物理寻址
- 帧：数据链路层传输单位

💻 代码示例

# socket编程示例 import socket # TCP客户端 def tcp_client(): s = socket.socket(socket.AF_INET, socket.SOCK_STREAM) s.connect(("example.com", 80)) s.send(b"GET / HTTP/1.1\r\nHost: example.com\r\n\r\n") data = s.recv(1024) s.close() return data # TCP服务器 def tcp_server(): s = socket.socket(socket.AF_INET, socket.SOCK_STREAM) s.bind(("0.0.0.0", 8080)) s.listen(5) while True: conn, addr = s.accept() data = conn.recv(1024) conn.send(b"Hello") conn.close()
