二分图判定（BFS/DFS染色法）

算法题 中等
标签：二分图, BFS, DFS, 染色法, 图论

📑 题目

如何判断一个图是否是二分图？

💡 参考答案

**二分图定义：**
图的顶点可分为两个不相交的集合U和V，使得每条边连接U中的一个顶点和V中的一个顶点。

**判定方法：染色法（BFS/DFS）**
1. 用两种颜色交替标记相邻节点
2. 如果发现相邻节点颜色相同，则不是二分图
3. 图可能不连通，需要对每个连通分量检查

**时间复杂度：** O(V + E)

**应用场景：**
1. 课程表问题（判断是否可以二部染色）
2. 匹配问题（匈牙利算法的基础）
3. 奇偶性判断

**关键结论：**
- 二分图不含奇数长度的环
- 树一定是二分图
- 一个图是二分图 当且仅当 不含奇环

💻 代码示例

from collections import deque from typing import List # ===== BFS染色法 ===== def is_bipartite_bfs(graph: List[List[int]]) -> bool: """BFS染色法判断二分图""" n = len(graph) color = [-1] * n # -1未染色, 0和1两种颜色 for start in range(n): if color[start] != -1: continue queue = deque([start]) color[start] = 0 while queue: node = queue.popleft() for neighbor in graph[node]: if color[neighbor] == -1: color[neighbor] = 1 - color[node] queue.append(neighbor) elif color[neighbor] == color[node]: return False return True # ===== DFS染色法 ===== def is_bipartite_dfs(graph: List[List[int]]) -> bool: """DFS染色法判断二分图""" n = len(graph) color = [-1] * n def dfs(node, c): color[node] = c for neighbor in graph[node]: if color[neighbor] == -1: if not dfs(neighbor, 1 - c): return False elif color[neighbor] == c: return False return True for i in range(n): if color[i] == -1: if not dfs(i, 0): return False return True # ===== LeetCode 785: 判断二分图 ===== # 测试 graph1 = [[1,3],[0,2],[1,3],[0,2]] # 二分图 graph2 = [[1,2,3],[0,1,3],[0,1,3],[0,1,2]] # 不是二分图 print(is_bipartite_bfs(graph1)) # True print(is_bipartite_bfs(graph2)) # False print(is_bipartite_dfs(graph1)) # True print(is_bipartite_dfs(graph2)) # False # ===== 可能的变体 ===== # 1. 返回二分图的两个集合 # 2. 判断是否可以重新排列使得相邻不同（等价于二分图） # 3. 二分图的最大匹配（匈牙利算法/ Hopcroft-Karp）
