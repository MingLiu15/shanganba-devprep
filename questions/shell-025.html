ssh/scp/rsync远程操作

标签：ssh, scp, rsync, 远程连接, 文件传输

📑 题目

如何使用ssh、scp、rsync进行远程操作？

💡 参考答案

**ssh - 远程登录：**
- `ssh user@host` - 远程登录
- `ssh -p port user@host` - 指定端口
- `ssh -i key.pem user@host` - 使用密钥
- `ssh user@host 'command'` - 执行远程命令

**scp - 远程复制：**
- `scp file user@host:/path` - 上传
- `scp user@host:/path/file .` - 下载
- `-r` - 递归复制目录
- `-P` - 指定端口

**rsync - 增量同步：**
- `-a` - 归档模式
- `-v` - 详细输出
- `-z` - 压缩传输
- `--delete` - 删除目标多余文件
- `--exclude` - 排除文件
