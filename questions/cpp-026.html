Lambda表达式的原理

C/C++ 中等
标签：Lambda, 闭包, 匿名函数, C++11

📑 题目

Lambda表达式是如何实现的？捕获列表的原理是什么？

💡 参考答案

实现原理：
编译器将lambda转换为匿名仿函数类（functor），重载operator()。

捕获列表：
- [=]：按值捕获所有外部变量（拷贝到成员变量）
- [&]：按引用捕获（存储引用）
- [this]：捕获当前对象指针
- [a, &b]：混合捕获

生成的类大致结构：
class Lambda_XXX {
Type1 captured_var1; // 按值捕获
Type2& captured_var2; // 按引用捕获
public:
ReturnType operator()(Params) const { /* body */ }
};

💻 代码示例

int x = 10, y = 20; // 等价于创建了一个带operator()的匿名类对象 auto lambda = [x, &y](int z) -> int { y++; // 修改原y return x + y + z; }; // 泛型lambda（C++14） auto add = [](auto a, auto b) { return a + b; }; // 使用 cout << lambda(5); // x(10) + y(21) + 5 = 36 cout << add(1, 2); cout << add(1.5, 2.5);
