运算符重载的规则

C/C++ 中等
标签：运算符重载, 成员函数, 友元函数

📑 题目

运算符重载有哪些规则和限制？

💡 参考答案

规则：
1. 不能创建新运算符，只能重载已有运算符
2. 不能改变运算符优先级和结合性
3. 不能改变操作数个数
4. :: ?: sizeof typeid 不能被重载
5. = () [] -> 必须作为成员函数重载
6. 流运算符<< >>通常作为友元函数

选择成员函数还是友元：
- 一元运算符、需要修改对象：成员函数
- 二元运算符、需要访问私有成员：友元函数

💻 代码示例

class Complex { double real, imag; public: // 成员函数重载一元运算符 Complex operator-() const { return Complex(-real, -imag); } // 友元函数重载二元运算符 friend Complex operator+(const Complex& a, const Complex& b) { return Complex(a.real + b.real, a.imag + b.imag); } };
