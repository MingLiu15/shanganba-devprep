序列化pickle与json

Python 中等
标签：pickle, json, 序列化, 反序列化, marshal, msgpack

📑 题目

比较pickle和json的区别及使用场景。

💡 参考答案

序列化：将对象转换为可存储/传输的字节流。

pickle：
- Python专用，支持几乎所有Python对象
- 二进制格式，效率高
- 不安全：不要加载不可信来源的数据
- 协议版本向前兼容

json：
- 通用文本格式，跨语言
- 只支持基本类型（dict, list, str, int, float, bool, None）
- 安全，人类可读
- 需要自定义编码器处理复杂对象

其他选项：
- marshal：Python内部使用，不保证兼容
- msgpack：二进制JSON，高效紧凑
- protobuf：Google协议缓冲区，强类型

💻 代码示例

import pickle import json data = {'name': 'Alice', 'age': 30, 'scores': [85, 90, 95]} # pickle序列化 pickled = pickle.dumps(data) restored = pickle.loads(pickled) # pickle到文件 with open('data.pkl', 'wb') as f: pickle.dump(data, f) # json序列化 json_str = json.dumps(data, indent=2) restored = json.loads(json_str) # json到文件 with open('data.json', 'w') as f: json.dump(data, f, indent=2) # 自定义JSON编码 class Person: def __init__(self, name, age): self.name = name self.age = age class PersonEncoder(json.JSONEncoder): def default(self, obj): if isinstance(obj, Person): return {'name': obj.name, 'age': obj.age} return super().default(obj) p = Person('Bob', 25) json_str = json.dumps(p, cls=PersonEncoder) # dataclass自动支持 from dataclasses import dataclass, asdict @dataclass class Point: x: float y: float json.dumps(asdict(Point(1, 2))) # pickle安全警告 # 不要pickle.load()不可信来源的数据！ # 可能被利用执行任意代码
