位域bit field

C/C++ 中等
标签：位域, bit field, 内存优化, 位操作

📑 题目

什么是位域？它有什么应用场景？

💡 参考答案

位域（Bit Field）：允许将结构体成员定义为指定比特位数。

语法：type name : bits;

应用场景：
1. 内存优化：当变量取值范围很小时
2. 硬件寄存器映射：精确控制内存布局
3. 协议解析：网络协议、文件格式

限制：
- 只能是整型（int, unsigned int, bool等）
- 不能取地址（&）
- 不能是静态成员
- 布局是实现定义的（大小端影响）

注意：
- 现代C++中，如果为了节省内存，考虑是否真的需要
- 位域操作可能比整型慢

💻 代码示例

// 位域结构体 struct Flags { unsigned int isActive : 1; // 1位 unsigned int priority : 3; // 3位（0-7） unsigned int category : 4; // 4位（0-15） unsigned int : 24; // 填充到32位 }; // sizeof(Flags) == 4（通常），而非12 // 网络协议头部 struct IPHeader { unsigned int version : 4; // IP版本 unsigned int ihl : 4; // 头部长度 unsigned int tos : 8; // 服务类型 unsigned int totalLength : 16; // 总长度 }; // 使用 Flags f; f.isActive = 1; f.priority = 5; // 注意：位域不能取地址 // int* p = &f.priority; // 错误！
