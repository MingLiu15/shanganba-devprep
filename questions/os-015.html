磁盘调度算法

操作系统 中等
标签：磁盘调度, SCAN, C-SCAN, 寻道, I/O

📑 题目

常见的磁盘调度算法有哪些？

💡 参考答案

磁盘访问时间 = 寻道时间 + 旋转延迟 + 传输时间
寻道时间是最大的因素。

1. FCFS（先来先服务）：
- 按请求顺序处理
- 简单但效率低，磁头移动距离大

2. SSTF（最短寻道时间优先）：
- 选择离当前磁头最近的请求
- 可能导致远处请求饥饿

3. SCAN（电梯算法）：
- 磁头沿一个方向移动到边界，再反向
- 类似电梯运行
- 对刚经过的位置不公平

4. C-SCAN（循环扫描）：
- 只在一个方向处理请求，到边界后快速回到起点
- 比SCAN更公平

5. LOOK/C-LOOK：
- 不必到边界，到最远的请求就反向
- SCAN和C-SCAN的优化版本

💻 代码示例

// SCAN电梯算法 void scan(int requests[], int n, int head, bool direction) { sort(requests, requests + n); int distance = 0; if (direction) { // 向大号方向 for (int i = 0; i < n; i++) { if (requests[i] >= head) { distance += abs(requests[i] - head); head = requests[i]; } } distance += abs(head - 199); // 到边界 head = 199; for (int i = n-1; i >= 0; i--) { if (requests[i] < 199) { distance += abs(head - requests[i]); head = requests[i]; } } } cout << "总寻道距离: " << distance << endl; }
