📑 题目

Python性能优化的常用方法有哪些？如何使用性能分析工具？

💡 参考答案

**性能分析工具：**
1. cProfile：内置性能分析器
2. line_profiler：逐行分析
3. memory_profiler：内存分析
4. timeit：代码计时

**优化策略：**
1. 算法优化：选择合适的数据结构和算法
2. 内置函数：使用map/filter代替循环
3. 列表推导：比普通循环快
4. 生成器：节省内存
5. 局部变量：访问比全局变量快
6. 字典查找：O(1)比列表O(n)快

**高级优化：**
1. NumPy：数值计算
2. Cython：编译为C
3. Numba：JIT编译
4. 多进程：利用多核

💻 代码示例

# 性能分析 import cProfile import pstats from io import StringIO def slow_function(): total = 0 for i in range(100000): total += i return total # 使用cProfile pr = cProfile.Profile() pr.enable() slow_function() pr.disable() s = StringIO() ps = pstats.Stats(pr, stream=s).sort_stats('cumulative') ps.print_stats(10) print(s.getvalue()) # 优化示例 # 1. 使用内置函数 # 慢 total = 0 for x in range(1000): total += x # 快 total = sum(range(1000)) # 2. 列表推导 vs 循环 # 慢 squares = [] for x in range(1000): squares.append(x ** 2) # 快 squares = [x ** 2 for x in range(1000)] # 3. 字典查找 vs 列表查找 # 慢 (O(n)) items = ['a', 'b', 'c', 'd', 'e'] if 'c' in items: pass # 快 (O(1)) items_set = set(items) if 'c' in items_set: pass # 4. 使用生成器节省内存 # 内存占用大 big_list = [x ** 2 for x in range(1000000)] # 内存友好 big_gen = (x ** 2 for x in range(1000000)) # 5. timeit计时 import timeit timeit.timeit('sum(range(1000))', number=10000)
