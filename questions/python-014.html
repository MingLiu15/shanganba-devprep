常用魔术方法

Python 中等
标签：面向对象, 魔术方法, 运算符重载

📑 题目

介绍Python中常用的魔术方法（__str__、__repr__、__call__、__getitem__等）及其作用。

💡 参考答案

魔术方法（双下划线方法/dunder methods）是Python中实现特殊行为的协议方法。

常用魔术方法分类：

字符串表示：
- __str__：str()和print()调用，面向用户的友好表示。
- __repr__：repr()调用，面向开发者的精确表示， ideally可以eval()还原对象。

调用与访问：
- __call__：使对象可以像函数一样被调用。
- __getitem__：实现obj[key]索引访问。
- __setitem__：实现obj[key] = value赋值。
- __getattr__：访问不存在的属性时触发。
- __getattribute__：访问任何属性时触发（慎用，容易死循环）。

比较运算：
- __eq__、__lt__、__gt__、__le__、__ge__、__ne__：实现==、<、>等比较。

算术运算：
- __add__、__sub__、__mul__等：实现+、-、*等运算符。
- __radd__等：实现右操作数运算（当左操作数不支持时）。

上下文管理：
- __enter__、__exit__：支持with语句。

容器协议：
- __len__、__iter__、__contains__：实现len()、迭代、in操作。

💻 代码示例

class Vector: def __init__(self, x, y): self.x, self.y = x, y def __repr__(self): return f'Vector({self.x}, {self.y})' def __str__(self): return f'({self.x}, {self.y})' def __add__(self, other): return Vector(self.x + other.x, self.y + other.y) def __call__(self, scalar): return Vector(self.x * scalar, self.y * scalar) def __getitem__(self, key): return (self.x, self.y)[key] v = Vector(3, 4) print(repr(v)) # Vector(3, 4) print(v + Vector(1, 1)) # Vector(4, 5) print(v(2)) # Vector(6, 8) print(v[0]) # 3
