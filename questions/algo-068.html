KMP字符串匹配算法

算法题 中等
标签：KMP, 字符串匹配, 前缀函数, next数组

📑 题目

KMP算法的原理是什么？如何构建next数组？

💡 参考答案

**KMP算法原理：**
利用已匹配的信息避免从头比较，时间复杂度O(m+n)。

**核心思想：**
- 当出现不匹配时，利用已匹配部分的信息，将模式串向右滑动尽可能远的距离
- 不需要回溯文本串指针

**next数组（前缀函数）：**
- next[i]表示模式串p[0:i]的最长相等真前缀和真后缀的长度
- 真前缀：不包含自身的所有前缀
- 真后缀：不包含自身的所有后缀

**构建next数组：**
- 双指针法，i遍历字符串，j记录当前匹配长度
- 时间复杂度O(m)

💻 代码示例

def build_next(pattern): m = len(pattern) next_arr = [0] * m j = 0 for i in range(1, m): while j > 0 and pattern[i] != pattern[j]: j = next_arr[j - 1] if pattern[i] == pattern[j]: j += 1 next_arr[i] = j return next_arr def kmp_search(text, pattern): if not pattern: return 0 n, m = len(text), len(pattern) next_arr = build_next(pattern) j = 0 positions = [] for i in range(n): while j > 0 and text[i] != pattern[j]: j = next_arr[j - 1] if text[i] == pattern[j]: j += 1 if j == m: positions.append(i - m + 1) j = next_arr[j - 1] return positions text = "ABABDABACDABABCABAB" pattern = "ABABCABAB" print(f"匹配位置: {kmp_search(text, pattern)}") # [10]
