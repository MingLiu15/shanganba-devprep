自定义内存分配器

C/C++ 困难
标签：allocator, 内存分配器, STL, 内存池

📑 题目

如何实现STL自定义内存分配器？

💡 参考答案

STL分配器：为容器提供内存分配/释放策略。

自定义分配器需要实现：
1. allocate(n)：分配n个元素的内存
2. deallocate(p, n)：释放内存
3. construct(p, args)：在p处构造对象
4. destroy(p)：析构p处的对象

应用场景：
- 内存池：减少频繁分配开销
- 共享内存：在特定内存区域分配
- 统计：跟踪内存使用情况

注意：
- C++17后allocator_traits提供默认实现
- 分配器应该是无状态的（stateless）

💻 代码示例

// 简单内存池分配器 template<typename T> class PoolAllocator { struct Block { char data[sizeof(T)]; Block* next; }; Block* freeList = nullptr; public: using value_type = T; T* allocate(size_t n) { if (n == 1 && freeList) { Block* block = freeList; freeList = freeList->next; return reinterpret_cast<T*>(block); } return static_cast<T*>(::operator new(n * sizeof(T))); } void deallocate(T* p, size_t n) { if (n == 1) { Block* block = reinterpret_cast<Block*>(p); block->next = freeList; freeList = block; } else { ::operator delete(p); } } }; // 使用 std::vector<int, PoolAllocator<int>> vec; vec.push_back(1); // 使用自定义分配器
