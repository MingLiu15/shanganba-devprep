static关键字的作用

C/C++ 中等
标签：static, 存储类, 作用域

📑 题目

说明C++中static关键字在不同场景下的作用。

💡 参考答案

1. 局部静态变量：延长生命周期至程序结束，只初始化一次
2. 全局静态变量/函数：限制作用域在当前文件（内部链接）
3. 类静态成员变量：所有对象共享，类外初始化
4. 类静态成员函数：属于类而非对象，只能访问静态成员

💻 代码示例

// 局部静态 void foo() { static int count = 0; // 只初始化一次 count++; } // 类静态 class MyClass { public: static int shared; // 声明 static void func() {} // 静态方法 }; int MyClass::shared = 0; // 定义初始化
