红黑树的特点

C/C++ 困难
标签：红黑树, 平衡二叉树, map, set

📑 题目

红黑树的性质是什么？为什么STL的map/set使用红黑树？

💡 参考答案

红黑树是自平衡二叉搜索树，通过颜色约束保持平衡。

五个性质：
1. 节点是红色或黑色
2. 根节点是黑色
3. 所有叶子（NIL）是黑色
4. 红色节点的子节点必须是黑色（无连续红节点）
5. 从任一节点到其叶子的所有路径包含相同数目的黑节点

优点：
- 插入、删除、查找都是O(log n)
- 最坏情况下也能保证性能
- 旋转次数较少（相比AVL树），适合频繁插入删除

STL的map/set使用红黑树原因：
- 需要有序性
- 插入删除查找效率均衡
- 实现相对简单

💻 代码示例

// 红黑树节点结构 struct RBNode { int key; bool red; // true=红, false=黑 RBNode *left, *right, *parent; }; // STL使用 std::map<int, string> m; // 红黑树 std::set<int> s; // 红黑树 std::unordered_map<int, string> um; // 哈希表
