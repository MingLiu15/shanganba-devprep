Python3新特性

Python 中等
标签：新特性, 类型注解, dataclass, walrus运算符

📑 题目

介绍Python 3中重要的新特性，包括类型注解、dataclass、海象运算符等。

💡 参考答案

Python 3引入了许多重要特性：

1. 类型注解（Type Hints，PEP 484）：
- 函数参数和返回值可以添加类型提示。
- 使用typing模块：List、Dict、Optional、Union、Tuple等。
- Python 3.10+使用 | 语法代替Union：int | None。
- 不影响运行时行为，主要用于静态检查（mypy）和IDE提示。

2. dataclass（PEP 557，Python 3.7+）：
- 自动生成__init__、__repr__、__eq__等方法。
- 减少样板代码。
- 支持默认值、不可变(frozen)、字段排序等。

3. 海象运算符（:=，PEP 572，Python 3.8+）：
- 赋值表达式，在表达式中同时赋值和使用。
- 减少重复计算，使代码更简洁。

4. f-string（Python 3.6+）：
- 格式化字符串字面量，比format()和%更简洁高效。
- Python 3.12+支持嵌套f-string和更灵活的表达式。

5. 其他重要特性：
- match-case模式匹配（Python 3.10+）。
- 异步生成器和异步推导式（Python 3.6+）。
- 字典合并运算符 |（Python 3.9+）。
- 结构化模式匹配（Python 3.10+）。

💻 代码示例

# 类型注解 from typing import Optional, List def greet(name: str, age: int = 0) -> str: return f'Hello, {name}, age {age}' # dataclass from dataclasses import dataclass, field @dataclass class User: name: str age: int = 0 tags: List[str] = field(default_factory=list) user = User('Tom', 25) print(user) # User(name='Tom', age=25, tags=[]) # 海象运算符 # 传统写法 # data = [1, 2, 3, 4, 5] # n = len(data) # if n > 3: # print(f'列表有{n}个元素') # 海象运算符写法 if (n := len([1,2,3,4,5])) > 3: print(f'列表有{n}个元素') # 字典合并 (Python 3.9+) d1 = {'a': 1} d2 = {'b': 2} merged = d1 | d2 # {'a': 1, 'b': 2} # match-case (Python 3.10+) # match command: # case 'quit': print('bye') # case ['load', filename]: print(f'加载 {filename}') # case _: print('未知命令')
