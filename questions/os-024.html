哲学家就餐问题

操作系统 中等
标签：哲学家就餐, 死锁, 避免死锁, 同步

📑 题目

什么是哲学家就餐问题？如何避免死锁？

💡 参考答案

问题描述：
- 5个哲学家围坐圆桌，每两人之间一根筷子
- 每个哲学家需要同时拿到左右两根筷子才能吃饭
- 拿不到就等待

死锁条件：
- 每人拿一根筷子后都在等另一根
- 形成循环等待

解决方案：
1. 资源有序分配：给筷子编号，必须先拿小编号
- 破坏循环等待条件

2. 最多允许4人同时拿筷子
- 破坏环路（至少一人能拿到两根）

3. 非对称获取：奇数号先拿左边，偶数号先拿右边
- 破坏循环等待

4. 和解方案：尝试拿不到就放下已拿的，随机等待后重试
- 类似CAS重试机制

💻 代码示例

// 方案1：资源有序分配 #include <pthread.h> pthread_mutex_t chopsticks[5]; void* philosopher(void* arg) { int i = *(int*)arg; int left = i; int right = (i + 1) % 5; // 先拿小编号（破坏循环等待） int first = min(left, right); int second = max(left, right); while (true) { think(); pthread_mutex_lock(&chopsticks[first]); pthread_mutex_lock(&chopsticks[second]); eat(); pthread_mutex_unlock(&chopsticks[second]); pthread_mutex_unlock(&chopsticks[first]); } }
