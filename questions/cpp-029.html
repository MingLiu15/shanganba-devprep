sizeof和strlen的区别

C/C++ 简单
标签：sizeof, strlen, 数组, 指针

📑 题目

sizeof和strlen有什么区别？

💡 参考答案

sizeof：
- 运算符，编译期计算
- 返回类型或对象占用的字节数
- 数组返回整个数组大小
- 指针返回指针本身大小（4或8字节）

strlen：
- 函数，运行期计算
- 返回C字符串长度（不含'\0'）
- 从首地址开始遍历直到遇到'\0'
- 只适用于以'\0'结尾的字符数组

常见陷阱：
- sizeof(数组名) = 数组总大小
- sizeof(数组参数) = 指针大小

💻 代码示例

char str[] = "hello"; char* p = str; cout << sizeof(str); // 6（包含'\0'） cout << strlen(str); // 5 cout << sizeof(p); // 4或8（指针大小） void func(char arr[]) { cout << sizeof(arr); // 4或8，退化为指针！ } int nums[10]; cout << sizeof(nums); // 40（int占4字节） cout << sizeof(nums[0]); // 4 cout << sizeof(nums)/sizeof(nums[0]); // 10，数组元素个数
