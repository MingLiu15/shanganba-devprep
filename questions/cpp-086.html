C++17/20 新特性概览

C/C++ 中等
标签：C++17, C++20, 结构化绑定, if constexpr, Concepts, 协程, Modules

📑 题目

C++17和C++20有哪些重要的新特性？

💡 参考答案

**C++17主要特性：**
- 结构化绑定（Structured Bindings）：auto [x, y] = pair;
- if constexpr：编译期条件
- 折叠表达式（Fold Expressions）：可变参数模板简化
- inline变量：头文件中定义变量
- std::optional、std::variant、std::any
- 字符串视图（std::string_view）
- 并行算法

**C++20主要特性：**
- Concepts：模板约束
- Ranges：范围库
- 协程（Coroutines）
- Modules：模块系统
- 三路比较运算符（<=>）
- 指定初始化（Designated Initializers）
- std::format
- constexpr虚函数、try-catch、new/delete

💻 代码示例

#include <iostream> #include <tuple> #include <optional> #include <variant> #include <string_view> // C++17 结构化绑定 std::tuple<int, double, std::string> get_data() { return {42, 3.14, "hello"}; } // C++17 if constexpr template<typename T> auto get_value(T t) { if constexpr (std::is_pointer_v<T>) { return *t; } else { return t; } } // C++17折叠表达式 template<typename... Args> auto sum(Args... args) { return (args + ...); // 一元右折叠 } // C++20 Concepts (需要支持C++20的编译器) // template<typename T> // concept Arithmetic = std::is_arithmetic_v<T>; // // template<Arithmetic T> // T add(T a, T b) { return a + b; } int main() { // 结构化绑定 auto [x, y, z] = get_data(); std::cout << x << ", " << y << ", " << z << std::endl; // 数组结构化绑定 int arr[3] = {1, 2, 3}; auto [a, b, c] = arr; // std::optional std::optional<int> maybe_value; maybe_value = 42; if (maybe_value) { std::cout << "Value: " << *maybe_value << std::endl; } // std::variant std::variant<int, double, std::string> var; var = 3.14; std::cout << std::get<double>(var) << std::endl; // std::string_view（零拷贝字符串引用） std::string str = "Hello World"; std::string_view sv(str.c_str() + 6, 5); std::cout << sv << std::endl; // World // 折叠表达式 std::cout << sum(1, 2, 3, 4, 5) << std::endl; // 15 // if constexpr int val = 42; std::cout << get_value(val) << std::endl; std::cout << get_value(&val) << std::endl; return 0; }
