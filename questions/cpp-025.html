右值引用和移动语义

C/C++ 困难
标签：右值引用, 移动语义, 完美转发, C++11

📑 题目

解释右值引用、移动语义和完美转发的概念。

💡 参考答案

右值引用（T&&）：绑定到临时对象（右值），可以安全地窃取资源。

移动语义：
- 移动构造函数：从临时对象转移资源，避免深拷贝
- 移动赋值运算符：转移资源
- std::move：将左值转为右值引用，启用移动

完美转发（std::forward）：
- 在模板中保持参数的值类别（左值/右值）
- 配合万能引用（T&&）使用

优势：大幅提升性能，避免不必要的拷贝

💻 代码示例

class String { char* data; public: // 移动构造函数 String(String&& other) noexcept { data = other.data; // 窃取指针 other.data = nullptr; // 置空源对象 } }; String createString() { return String("hello"); } String s = createString(); // 移动构造，无拷贝 // 完美转发 template<typename T> void wrapper(T&& arg) { foo(std::forward<T>(arg)); // 保持原值类别 }
