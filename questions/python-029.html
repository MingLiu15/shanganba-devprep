functools常用工具

Python 中等
标签：标准库, functools, lru_cache, 装饰器

📑 题目

介绍Python functools模块中常用的工具函数，包括lru_cache、partial、wraps等。

💡 参考答案

functools是Python标准库中提供高阶函数操作的模块。

常用工具：

1. lru_cache（最近最少使用缓存）：
- 缓存函数调用结果，相同参数直接返回缓存值。
- @lru_cache(maxsize=128)：设置缓存大小。
- @lru_cache(maxsize=None)：无限制缓存。
- 适合纯函数（相同输入总是返回相同输出）。
- cache_info()查看缓存统计，cache_clear()清除缓存。

2. partial（偏函数）：
- 固定函数的部分参数，返回新的可调用对象。
- functools.partial(func, *args, **kwargs)。
- 简化重复调用相同参数的场景。

3. wraps（装饰器工具）：
- 保留被装饰函数的元信息（__name__、__doc__、__module__等）。
- 装饰器中必须使用，否则被装饰函数的元信息会丢失。

4. reduce（归约）：
- functools.reduce(func, iterable[, initializer])。
- 将可迭代对象归约为单个值。

5. singledispatch（单分派泛型函数）：
- 根据第一个参数的类型选择不同的实现。

6. total_ordering：
- 只需定义__eq__和一个比较方法，自动生成其余比较方法。

💻 代码示例

from functools import lru_cache, partial, wraps, reduce # lru_cache @lru_cache(maxsize=128) def fibonacci(n): if n < 2: return n return fibonacci(n-1) + fibonacci(n-2) print(fibonacci(100)) # 瞬间完成 print(fibonacci.cache_info()) # CacheInfo(hits=98, misses=101, maxsize=128, currsize=101) # partial power = partial(pow, exp=2) print(power(5)) # 25 print(power(10)) # 100 # wraps def my_decorator(func): @wraps(func) def wrapper(*args, **kwargs): print(f'调用 {func.__name__}') return func(*args, **kwargs) return wrapper @my_decorator def hello(name): """打招呼""" return f'Hello, {name}' print(hello.__name__) # hello（保留了原名） print(hello.__doc__) # 打招呼 # reduce print(reduce(lambda x, y: x + y, [1, 2, 3, 4, 5])) # 15
