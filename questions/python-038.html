LEGB变量查找规则

Python 中等
标签：LEGB, 变量作用域, 命名空间, 闭包, global

📑 题目

解释Python的LEGB变量查找规则。

💡 参考答案

LEGB：Python变量查找的顺序规则。

L - Local（局部）：函数内部
E - Enclosing（嵌套）：嵌套函数的上一层函数
G - Global（全局）：模块级别
B - Built-in（内置）：Python内置名称

查找顺序：Local → Enclosing → Global → Built-in

注意事项：
1. 赋值操作默认创建局部变量
2. global声明使用全局变量
3. nonlocal声明使用嵌套变量
4. 只读访问会按LEGB查找，但赋值需要显式声明

💻 代码示例

x = 'global' # G def outer(): x = 'enclosing' # E def inner(): x = 'local' # L print(x) # 输出 'local' inner() print(x) # 输出 'enclosing' outer() print(x) # 输出 'global' # global和nonlocal count = 0 def increment(): global count # 声明使用全局变量 count += 1 def outer2(): x = 10 def inner2(): nonlocal x # 声明使用嵌套变量 x += 1 inner2() print(x) # 11 # 常见错误 def wrong(): print(x) # 可以，按LEGB找到global x = 5 # 错误！赋值视为局部变量，但上面已经使用了 # 正确做法 def correct(): global x print(x) x = 5
