构造函数和析构函数的调用顺序

C/C++ 中等
标签：构造函数, 析构函数, 继承, 成员对象

📑 题目

有继承关系时，构造函数和析构函数的调用顺序是什么？

💡 参考答案

构造函数调用顺序：
1. 基类构造函数（按继承声明顺序）
2. 成员对象构造函数（按声明顺序）
3. 自身构造函数

析构函数调用顺序（相反）：
1. 自身析构函数
2. 成员对象析构函数（与声明相反顺序）
3. 基类析构函数（与继承相反顺序）

虚继承时，虚基类构造函数最先调用

💻 代码示例

class A { A() { cout << "A"; } ~A() { cout << "~A"; } }; class B { B() { cout << "B"; } ~B() { cout << "~B"; } }; class C : public A { B b; public: C() { cout << "C"; } // 输出: A B C ~C() { cout << "~C"; } // 输出: ~C ~B ~A };
