工厂模式

C/C++ 中等
标签：设计模式, 工厂模式, 创建型模式

📑 题目

解释工厂模式的作用和实现方式。

💡 参考答案

工厂模式：定义一个创建对象的接口，让子类决定实例化哪个类。将对象的创建与使用分离。

类型：
1. 简单工厂：一个工厂类根据参数创建不同产品
2. 工厂方法：每个产品对应一个工厂，继承抽象工厂
3. 抽象工厂：创建相关的产品族

优点：
- 解耦：客户端不需要知道具体产品类
- 扩展性好：新增产品只需添加工厂
- 符合开闭原则

缺点：
- 类数量增加
- 增加系统复杂度

💻 代码示例

// 抽象产品 class Button { public: virtual void render() = 0; }; // 具体产品 class WindowsButton : public Button { void render() override { /* Windows风格 */ } }; class MacButton : public Button { void render() override { /* Mac风格 */ } }; // 工厂 class ButtonFactory { public: virtual Button* createButton() = 0; }; class WindowsFactory : public ButtonFactory { Button* createButton() override { return new WindowsButton(); } }; // 使用 ButtonFactory* factory = new WindowsFactory(); Button* btn = factory->createButton(); btn->render();
