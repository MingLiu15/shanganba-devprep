Linux网络工具：curl/wget/nc

Shell脚本 中等
标签：curl, wget, nc, netcat, 网络工具, HTTP

📑 题目

curl、wget和nc（netcat）的常用用法？

💡 参考答案

**curl：**
- 强大的HTTP客户端
- 支持各种协议（HTTP、FTP、SMTP等）
- 常用于API测试、文件下载

**wget：**
- 专注于文件下载
- 支持断点续传
- 递归下载网站

**nc（netcat）：**
- 网络瑞士军刀
- 端口扫描、文件传输
- 创建简单服务器
- 网络调试

💻 代码示例

# curl常用命令 curl_commands = """ # GET请求 curl https://api.example.com/users # POST请求 curl -X POST -H "Content-Type: application/json" -d '{"name":"Alice"}' https://api.example.com/users # 下载文件 curl -O https://example.com/file.zip curl -o myfile.zip https://example.com/file.zip # 跟随重定向 curl -L https://bit.ly/xxx # 显示响应头 curl -I https://example.com """ # wget常用命令 wget_commands = """ # 下载文件 wget https://example.com/file.zip # 断点续传 wget -c https://example.com/large-file.zip # 后台下载 wget -b https://example.com/file.zip # 递归下载网站 wget -r -l 2 https://example.com """ # nc常用命令 nc_commands = """ # 端口扫描 nc -zv localhost 20-80 # 创建简单服务器 nc -l 8080 # 文件传输（接收方） nc -l 1234 > received_file # 文件传输（发送方） nc receiver_ip 1234 < file_to_send # 作为代理 nc -l 8080 | nc target_host 80 """ print(curl_commands) print(wget_commands) print(nc_commands)
