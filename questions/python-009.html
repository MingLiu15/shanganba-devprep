is与==的区别

Python 简单
标签：基础语法, 比较运算, 对象模型

📑 题目

解释Python中is和==运算符的区别，以及各自的适用场景。

💡 参考答案

==（相等比较）：比较两个对象的值是否相等，调用对象的__eq__()方法。判断的是值层面的等价性。

is（身份比较）：比较两个对象的内存地址是否相同，即是否是同一个对象。判断的是身份层面的同一性。

关键区别：
- ==比较值，is比较地址（id()）。
- a == b 为True不代表 a is b 为True（不同对象可以有相同的值）。
- a is b 为True则 a == b 一定为True（同一对象值必然相同）。

适用场景：
- 比较值是否相等用==：如字符串内容、数字大小、列表元素。
- 判断是否是同一对象用is：如与None比较（推荐用is None而非==None）、单例判断。

Python小整数缓存：CPython对-5到256的整数进行了缓存，相同值的小整数is比较为True。字符串也有类似的intern机制，但不可依赖。

注意：浮点数比较推荐使用math.isclose()而非==，因为浮点精度问题。

💻 代码示例

# == vs is a = [1, 2, 3] b = [1, 2, 3] print(a == b) # True，值相等 print(a is b) # False，不同对象 # 小整数缓存 c = 256 d = 256 print(c is d) # True，在缓存范围内 e = 257 f = 257 print(e is f) # False，超出缓存范围 # None比较 x = None print(x is None) # True（推荐写法） print(x == None) # True（不推荐） # 字符串intern s1 = 'hello' s2 = 'hello' print(s1 is s2) # True（短字符串会被intern）
