*args和**kwargs的作用

Python 简单
标签：基础语法, 函数参数, 可变参数

📑 题目

解释Python中*args和**kwargs的作用、使用场景及注意事项。

💡 参考答案

*args和**kwargs是Python函数的可变参数语法，用于处理不确定数量的参数。

*args：接收任意数量的位置参数，以元组(tuple)形式传递。args只是约定名称，可以用其他名称。
**kwargs：接收任意数量的关键字参数，以字典(dict)形式传递。kwargs也只是约定名称。

使用场景：
- 装饰器中包装函数时保持原函数签名。
- 函数需要灵活接受不同数量参数时。
- 子类调用父类方法时传递参数。

注意事项：
- *args必须在**kwargs之前。
- 函数参数顺序：普通参数、*args、默认参数、**kwargs。
- 单独使用*可以强制后续参数必须以关键字形式传入。
- 可以用*解包列表/元组，用**解包字典传参。

💻 代码示例

def func(a, b, *args, key='default', **kwargs): print(f'a={a}, b={b}') print(f'args={args}') print(f'key={key}') print(f'kwargs={kwargs}') func(1, 2, 3, 4, 5, key='hello', x=10, y=20) # a=1, b=2 # args=(3, 4, 5) # key=hello # kwargs={'x': 10, 'y': 20} # 解包传参 args = [1, 2, 3] kwargs = {'a': 10, 'b': 20} def show(x, y, z, a, b): print(x, y, z, a, b) show(*args, **kwargs) # 1 2 3 10 20
