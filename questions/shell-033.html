awk自定义函数与关联数组

Shell脚本 困难
标签：awk, 自定义函数, 关联数组, 数组排序, 高级用法

📑 题目

awk的自定义函数和关联数组有哪些高级用法？

💡 参考答案

**自定义函数：**
- function name(params) { statements; return value }
- 支持递归调用
- 可以定义在BEGIN块之前

**关联数组：**
- 键可以是字符串或数字
- 自动创建，无需声明
- 支持in运算符检查键存在
- delete array[key]删除元素

**数组操作：**
- asort()：值排序
- asorti()：键排序
- split()：字符串分割
- PROCINFO["sorted_in"]：控制遍历顺序

**高级技巧：**
- 二维数组模拟
- 数组去重
- 动态键构建

💻 代码示例

awk_examples = """ # 定义函数 function max(a, b) { return (a > b) ? a : b } { print max($1, $2) } # 关联数组 { count[$1]++ } END { for (name in count) print name, count[name] } # asort排序 END { n = asort(count, s); for(i=1; i<=n; i++) print s[i] } # split函数 n = split($1, parts, ":"); for(i=1; i<=n; i++) print parts[i] # 统计日志 awk '{count[$7]++} END { for(url in count) print count[url], url}' access.log """ print(awk_examples)
