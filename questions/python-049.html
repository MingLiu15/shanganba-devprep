Celery分布式任务队列

Python 困难
标签：Celery, 分布式, 任务队列, 异步任务, 定时任务

📑 题目

Celery是什么？如何配置和使用Celery处理异步任务？

💡 参考答案

**Celery核心概念：**
1. Broker：消息代理（Redis/RabbitMQ）
2. Worker：任务执行者
3. Task：异步任务函数
4. Result Backend：结果存储
5. Beat：定时任务调度器

**任务类型：**
- apply_async()：异步执行
- delay()：简化的异步调用
- s()/si()：签名（任务模板）

**任务状态：**
PENDING → STARTED → SUCCESS/FAILURE

**常用配置：**
- CELERY_BROKER_URL
- CELERY_RESULT_BACKEND
- CELERY_TASK_SERIALIZER
- CELERY_TIMEZONE

💻 代码示例
