模板特化与偏特化

C/C++ 困难
标签：模板特化, 偏特化, 全特化, 模板元编程

📑 题目

什么是模板特化和偏特化？有什么区别和应用场景？

💡 参考答案

模板特化：为特定类型提供专门的模板实现。

全特化（Full Specialization）：
- 为特定类型完全定制实现
- 所有模板参数都被指定
- 原模板的所有代码都被替换

偏特化（Partial Specialization）：
- 只特化部分模板参数
- 保留部分泛型能力
- 类模板支持，函数模板不支持（C++20前）

应用场景：
1. 全特化：为特定类型提供完全不同的实现（如vector<bool>）
2. 偏特化：为某一类类型提供优化（如指针类型、数组类型）

优先级：全特化 > 偏特化 > 主模板

💻 代码示例

// 主模板 template<typename T> class Container { public: void print() { cout << "通用容器" << endl; } }; // 全特化 - 为int类型完全定制 template<> class Container<int> { public: void print() { cout << "int专用容器" << endl; } }; // 偏特化 - 为指针类型优化 template<typename T> class Container<T*> { public: void print() { cout << "指针容器" << endl; } }; // 偏特化 - 为数组类型 template<typename T, size_t N> class Container<T[N]> { public: void print() { cout << "数组容器，大小" << N << endl; } }; // 使用 Container<double> c1; // 通用 Container<int> c2; // 全特化 Container<int*> c3; // 偏特化（指针） Container<int[10]> c4; // 偏特化（数组）
