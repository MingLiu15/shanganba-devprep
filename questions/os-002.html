进程调度算法

操作系统 中等
标签：调度, FCFS, SJF, RR, 优先级

📑 题目

常见的进程调度算法有哪些？各有什么优缺点？

💡 参考答案

1. 先来先服务（FCFS）：
- 按到达顺序执行，非抢占式
- 优点：简单公平
- 缺点：短作业等待时间长，平均等待时间长

2. 短作业优先（SJF）：
- 选择执行时间最短的进程
- 优点：平均等待时间最短
- 缺点：长作业可能饥饿，难以预知执行时间

3. 时间片轮转（RR）：
- 每个进程分配固定时间片，用完切换
- 优点：响应时间快，适合分时系统
- 缺点：时间片太大退化为FCFS，太小切换开销大

4. 优先级调度：
- 按优先级选择进程
- 优点：灵活
- 缺点：低优先级进程可能饥饿

5. 多级反馈队列：
- 多个队列，优先级递减，时间片递增
- 综合以上算法优点，现代操作系统常用

💻 代码示例

// 时间片轮转模拟 struct Process { int pid, burst_time, remaining_time; }; void roundRobin(Process p[], int n, int quantum) { queue<int> ready; for (int i = 0; i < n; i++) ready.push(i); while (!ready.empty()) { int idx = ready.front(); ready.pop(); int exec = min(quantum, p[idx].remaining_time); p[idx].remaining_time -= exec; if (p[idx].remaining_time > 0) ready.push(idx); // 未完成，放回队尾 } }
