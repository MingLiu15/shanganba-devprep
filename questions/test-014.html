测试用例设计方法：边界值分析

自动化测试 简单
标签：测试设计, 边界值, 黑盒测试, 测试用例

📑 题目

什么是边界值分析法？如何应用？

💡 参考答案

**边界值分析定义：**
经验表明，大量错误发生在输入范围的边界上。边界值分析是对等价类划分的补充，重点测试边界值及其附近值。

**边界值选取原则：**
对于范围[a, b]：
- 最小值 a
- 略大于最小值 a+
- 正常值（范围内任意值）
- 略小于最大值 b-
- 最大值 b

**边界值类型：**
1. **上点**：边界上的点（a, b）
2. **离点**：离边界最近的点（a-1, b+1）
3. **内点**：范围内的点

**示例：年龄输入（18-60岁）**
- 边界值：17, 18, 19, 30, 59, 60, 61

💻 代码示例

# 边界值分析测试示例 import pytest def validate_age(age): """验证年龄：18-60岁""" if not isinstance(age, int): raise TypeError("年龄必须是整数") if age < 18: raise ValueError("年龄不能小于18岁") if age > 60: raise ValueError("年龄不能超过60岁") return True class TestAgeBoundary: # 边界值测试 def test_boundary_below_min(self): """下边界-1: 17岁（无效）""" with pytest.raises(ValueError): validate_age(17) def test_boundary_min(self): """下边界: 18岁（有效）""" assert validate_age(18) def test_boundary_above_min(self): """下边界+1: 19岁（有效）""" assert validate_age(19) def test_normal_value(self): """正常值: 30岁（有效）""" assert validate_age(30) def test_boundary_below_max(self): """上边界-1: 59岁（有效）""" assert validate_age(59) def test_boundary_max(self): """上边界: 60岁（有效）""" assert validate_age(60) def test_boundary_above_max(self): """上边界+1: 61岁（无效）""" with pytest.raises(ValueError): validate_age(61) # 使用pytest.mark.parametrize参数化 @pytest.mark.parametrize("age,expected", [ (17, False), # 下边界-1 (18, True), # 下边界 (60, True), # 上边界 (61, False), # 上边界+1 ]) def test_age_boundaries(self, age, expected): if expected: assert validate_age(age) else: with pytest.raises(ValueError): validate_age(age)
