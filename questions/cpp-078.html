C++11 Lambda表达式详解

C/C++ 中等
标签：C++11, Lambda, 闭包, 函数对象, 现代C++

📑 题目

C++11 Lambda表达式的语法和捕获方式有哪些？

💡 参考答案

**Lambda语法：**
[capture](params) -> ret { body }

**捕获方式：**
- []：不捕获任何外部变量
- [=]：值捕获所有外部变量
- [&]：引用捕获所有外部变量
- [this]：捕获当前对象指针
- [x, &y]：混合捕获（x值捕获，y引用捕获）

**关键特性：**
1. Lambda本质是匿名函数对象（编译器生成仿函数类）
2. 捕获变量存储在闭包中
3. mutable允许修改值捕获的变量
4. 可以转换为std::function

**C++14泛型Lambda：**
auto lambda = [](auto x, auto y) { return x + y; };

💻 代码示例

#include <iostream> #include <vector> #include <algorithm> #include <functional> int main() { int x = 10, y = 20; // 1. 基本Lambda auto basic = []() { std::cout << "Hello Lambda" << std::endl; }; basic(); // 2. 带参数和返回值 auto add = [](int a, int b) -> int { return a + b; }; std::cout << add(3, 4) << std::endl; // 7 // 3. 捕获方式 // 值捕获（默认const，不可修改） auto capture_val = [=]() { return x + y; }; // 引用捕获 auto capture_ref = [&]() { x++; y++; return x + y; }; // 混合捕获 auto capture_mix = [x, &y]() { y++; return x + y; }; // mutable：允许修改值捕获的变量 auto mutable_lambda = [x]() mutable { x++; return x; }; std::cout << mutable_lambda() << std::endl; // 11 std::cout << x << std::endl; // 10（原值不变） // 4. 泛型Lambda（C++14） auto generic = [](auto a, auto b) { return a + b; }; std::cout << generic(1, 2) << std::endl; // int std::cout << generic(1.5, 2.5) << std::endl; // double // 5. 实际应用：STL算法 std::vector<int> vec = {3, 1, 4, 1, 5, 9}; // 排序 std::sort(vec.begin(), vec.end(), [](int a, int b) { return a > b; // 降序 }); // 查找 auto it = std::find_if(vec.begin(), vec.end(), [](int n) { return n > 4; }); // 6. 存储在std::function中 std::function<int(int, int)> func = [](int a, int b) { return a * b; }; return 0; }
