嵌套类与局部类

C/C++ 中等
标签：嵌套类, 局部类, 内部类, 类作用域

📑 题目

什么是嵌套类和局部类？它们有什么特点？

💡 参考答案

嵌套类（Nested Class）：定义在另一个类内部的类。

特点：
- 是外围类的成员，受访问控制（public/private）
- 可以访问外围类的私有静态成员
- 不能自动访问外围类的非静态成员（需通过对象）

局部类（Local Class）：定义在函数内部的类。

特点：
- 只在定义它的函数内可见
- 不能定义静态成员
- 可以访问外围函数的局部变量（C++20前不能，C++20后有限支持）

应用场景：
- 嵌套类：实现细节隐藏（如迭代器实现）
- 局部类：回调函数的简单实现

💻 代码示例

// 嵌套类示例 class Outer { static int staticData; int data; public: class Inner { // 默认private public: void func() { // 可以访问staticData // 不能直接访问data（需要Outer对象） } }; }; // 使用 Outer::Inner obj; // 如果Inner是public // 局部类示例 void process() { int localVar = 10; class Local { public: void print() { // C++20前不能访问localVar std::cout << "Local class\n"; } }; Local l; l.print(); } // STL迭代器是嵌套类的典型应用 std::vector<int>::iterator it; // iterator是vector的嵌套类
