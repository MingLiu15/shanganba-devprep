C++11 右值引用与移动语义

C/C++ 困难
标签：C++11, 右值引用, 移动语义, 完美转发, 现代C++

📑 题目

什么是右值引用？移动语义和完美转发的原理是什么？

💡 参考答案

**左值vs右值：**
- 左值：有名字、可取地址、可赋值（如变量）
- 右值：临时对象、字面量、即将销毁的值

**右值引用（&&）：**
- 绑定到右值的引用
- 允许"窃取"资源而非拷贝
- std::move将左值转为右值引用

**移动语义：**
- 移动构造函数：参数为右值引用
- 移动赋值运算符
- 转移资源所有权，避免深拷贝

**完美转发（Perfect Forwarding）：**
- std::forward保持参数的左右值属性
- 配合万能引用（T&&）使用
- 实现参数的完美传递

**万能引用（Universal Reference）：**
- 模板参数T&&或auto&&
- 既能绑定左值也能绑定右值

💻 代码示例

#include <iostream> #include <vector> #include <string> #include <utility> class MyString { char* data; size_t len; public: // 构造函数 MyString(const char* str) { len = strlen(str); data = new char[len + 1]; strcpy(data, str); std::cout << "Constructor: " << data << std::endl; } // 拷贝构造函数（深拷贝） MyString(const MyString& other) { len = other.len; data = new char[len + 1]; strcpy(data, other.data); std::cout << "Copy Constructor: " << data << std::endl; } // 移动构造函数（转移资源） MyString(MyString&& other) noexcept { data = other.data; // 窃取指针 len = other.len; other.data = nullptr; // 置空源对象 other.len = 0; std::cout << "Move Constructor: " << data << std::endl; } // 拷贝赋值 MyString& operator=(const MyString& other) { if (this != &other) { delete[] data; len = other.len; data = new char[len + 1]; strcpy(data, other.data); } std::cout << "Copy Assignment" << std::endl; return *this; } // 移动赋值 MyString& operator=(MyString&& other) noexcept { if (this != &other) { delete[] data; data = other.data; len = other.len; other.data = nullptr; other.len = 0; } std::cout << "Move Assignment" << std::endl; return *this; } ~MyString() { if (data) { std::cout << "Destructor: " << data << std::endl; delete[] data; } } }; // 完美转发模板 template<typename T> void wrapper(T&& arg) { // T&&是万能引用，arg是左值 // 使用forward保持原始左右值属性 foo(std::forward<T>(arg)); } void foo(const std::string& s) { std::cout << "Lvalue: " << s << std::endl; } void foo(std::string&& s) { std::cout << "Rvalue: " << s << std::endl; } int main() { MyString s1("Hello"); // 拷贝构造 MyString s2 = s1; // 移动构造（使用临时对象） MyString s3 = MyString("World"); // std::move强制转为右值 MyString s4 = std::move(s1); // 完美转发示例 std::string str = "test"; wrapper(str); // 调用Lvalue版本 wrapper("temporary"); // 调用Rvalue版本 return 0; }
