sed分组捕获与保持空间

Shell脚本 困难
标签：sed, 分组捕获, 保持空间, 高级替换, 正则表达式

📑 题目

sed的正则分组捕获和保持空间如何使用？

💡 参考答案

**分组捕获（Capture Groups）：**
- \(pattern\)：定义捕获组
- \1, \2：引用捕获内容
- 支持最多9个分组

**保持空间（Hold Space）：**
- 模式空间（Pattern Space）：当前行
- 保持空间（Hold Space）：额外存储
- h/H：复制/追加到保持空间
- g/G：复制/追加到模式空间
- x：交换模式空间和保持空间

**应用场景：**
- 多行模式匹配
- 复杂文本转换
- 行列转置

💻 代码示例

# sed分组捕获示例 sed_examples = """ # 基本分组替换：交换first和last sed -E 's/([A-Za-z]+), ([A-Za-z]+)/\\2 \\1/' file.txt # 提取域名 echo "user@example.com" | sed -E 's/.*@([^.]+)\\..*/\\1/' # 去除HTML标签 cat index.html | sed -E 's/<[^>]+>//g' # 多次分组引用 sed -E 's/([0-9]+)-([0-9]+)-([0-9]+)/(\\1) \\2-\\3/' file # 保持空间示例：反转行 sed -n '{1!G; h; $p}' file.txt # 保持空间示例：行列转置 sed -n '{H; g; s/\\n//g; p}' file.txt # 复杂示例：处理多行记录 sed -E ':a; N; s/([0-9]+)\\n([0-9]+)/\\1 \\2/; ba' file """ print(sed_examples)
