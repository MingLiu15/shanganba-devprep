C++11 初始化列表与委托构造

C/C++ 中等
标签：C++11, 初始化列表, 委托构造, 统一初始化, 现代C++

📑 题目

C++11的列表初始化、委托构造函数和继承构造函数是什么？

💡 参考答案

**统一初始化（列表初始化）：**
- 使用{}进行初始化
- 防止窄化转换（narrowing conversion）
- 可用于所有类型（基本类型、类、数组、STL容器）
- std::initializer_list支持初始化列表参数

**委托构造函数：**
- 一个构造函数调用同类其他构造函数
- 减少代码重复
- 委托构造函数体在目标构造函数之后执行

**继承构造函数（C++11）：**
- using Base::Base;
- 子类继承基类的构造函数
- 避免重复编写转发构造函数

💻 代码示例

#include <iostream> #include <vector> #include <initializer_list> class MyClass { int x, y, z; public: // 目标构造函数 MyClass(int a, int b, int c) : x(a), y(b), z(c) { std::cout << "Target Constructor" << std::endl; } // 委托构造函数 MyClass() : MyClass(0, 0, 0) { std::cout << "Delegating Constructor" << std::endl; } MyClass(int a) : MyClass(a, 0, 0) { std::cout << "Single arg delegating" << std::endl; } }; // 继承构造函数 class Base { public: Base(int x) { std::cout << "Base(int)" << std::endl; } Base(int x, int y) { std::cout << "Base(int, int)" << std::endl; } Base(const std::string& s) { std::cout << "Base(string)" << std::endl; } }; class Derived : public Base { public: // 继承基类所有构造函数 using Base::Base; // 可以添加自己的构造函数 Derived(double d) : Base(static_cast<int>(d)) { std::cout << "Derived(double)" << std::endl; } }; // initializer_list示例 class MyVector { std::vector<int> data; public: MyVector(std::initializer_list<int> list) : data(list) { std::cout << "Initialized with " << list.size() << " elements" << std::endl; } }; int main() { // 统一初始化 int a{42}; // 直接初始化 int b = {42}; // 拷贝初始化 // 防止窄化转换 // int c{3.14}; // 编译错误：double到int是窄化 // 数组初始化 int arr[]{1, 2, 3, 4, 5}; // STL容器 std::vector<int> vec{1, 2, 3, 4, 5}; // 自定义类 MyVector mv{1, 2, 3, 4, 5}; // 委托构造 MyClass obj1; // 调用委托构造 MyClass obj2(10); // 调用单参数委托构造 MyClass obj3(1, 2, 3); // 直接调用目标构造 // 继承构造 Derived d1(42); // 使用继承的Base(int) Derived d2(1, 2); // 使用继承的Base(int, int) Derived d3("hello"); // 使用继承的Base(string) Derived d4(3.14); // 使用Derived(double) return 0; }
