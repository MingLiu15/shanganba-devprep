systemctl与Linux服务管理

Shell脚本 中等
标签：systemctl, systemd, 服务管理, Linux, 守护进程

📑 题目

如何使用systemctl管理服务？如何编写systemd服务单元文件？

💡 参考答案

**systemctl常用命令：**
- systemctl start/stop/restart service
- systemctl enable/disable service（开机启动）
- systemctl status service
- systemctl list-units --type=service

**服务单元文件（.service）：**
- [Unit]：描述和依赖
- [Service]：服务配置
- Type：simple/forking/oneshot
- ExecStart：启动命令
- Restart：自动重启策略
- [Install]：安装配置
- WantedBy：目标运行级别

💻 代码示例

# systemd服务单元文件示例 service_file = """ [Unit] Description=My Application Service After=network.target [Service] Type=simple User=www-data WorkingDirectory=/opt/myapp ExecStart=/usr/bin/python3 /opt/myapp/main.py Restart=always RestartSec=10 [Install] WantedBy=multi-user.target """ # 常用命令 commands = """ # 重新加载配置 systemctl daemon-reload # 启动服务 systemctl start myapp # 开机启动 systemctl enable myapp # 查看日志 journalctl -u myapp -f # 查看所有服务 systemctl list-units --type=service --state=running """ print(service_file) print(commands)
