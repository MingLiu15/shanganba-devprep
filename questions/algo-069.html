字符串哈希（Rolling Hash）

算法题 中等
标签：字符串哈希, Rabin-Karp, 滚动哈希, 哈希冲突

📑 题目

什么是字符串哈希？如何解决哈希冲突？

💡 参考答案

**字符串哈希原理：**
将字符串映射为一个整数（哈希值），利用滚动哈希在O(1)时间内比较子串。

**多项式哈希：**
hash(s) = s[0]*base^(n-1) + s[1]*base^(n-2) + ... + s[n-1]*base^0

**滚动哈希：**
- 预处理前缀哈希
- 利用前缀哈希快速计算任意子串哈希
- 时间复杂度O(1)查询

**哈希冲突处理：**
1. 双哈希：使用两个不同模数
2. 自然溢出：利用unsigned long long溢出
3. 选择大质数作为模数

💻 代码示例

class StringHash: def __init__(self, s, base=131, mod=10**18 + 9): self.n = len(s) self.base = base self.mod = mod self.prefix = [0] * (self.n + 1) self.power = [1] * (self.n + 1) for i in range(self.n): self.power[i + 1] = (self.power[i] * base) % mod self.prefix[i + 1] = (self.prefix[i] * base + ord(s[i])) % mod def get_hash(self, left, right): res = (self.prefix[right + 1] - self.prefix[left] * self.power[right - left + 1]) % self.mod return res if res >= 0 else res + self.mod # Rabin-Karp字符串匹配 def rabin_karp(text, pattern): if len(pattern) > len(text): return [] ph = StringHash(pattern) th = StringHash(text) pattern_hash = ph.get_hash(0, len(pattern) - 1) positions = [] for i in range(len(text) - len(pattern) + 1): if th.get_hash(i, i + len(pattern) - 1) == pattern_hash: if text[i:i + len(pattern)] == pattern: positions.append(i) return positions print(f"匹配位置: {rabin_karp('ABABDABACDABABCABAB', 'ABABCABAB')}")
