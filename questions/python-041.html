typing模块高级用法

Python 困难
标签：typing, 类型注解, TypeVar, Generic, Protocol, 类型提示

📑 题目

介绍typing模块的高级用法。

💡 参考答案

typing：Python类型注解支持模块，用于静态类型检查。

高级特性：
1. TypeVar：泛型类型变量
2. Generic：自定义泛型类
3. Protocol：结构子类型（鸭子类型形式化）
4. Union/Optional：联合类型
5. Callable：函数类型
6. Type：类本身类型

类型检查工具：
- mypy：静态类型检查器
- pyright：微软的类型检查器
- IDE支持：PyCharm、VSCode等

💻 代码示例

from typing import TypeVar, Generic, Protocol, Union, Optional, Callable, List, Dict # TypeVar - 泛型变量 T = TypeVar('T') K = TypeVar('K') V = TypeVar('V') def first(items: List[T]) -> Optional[T]: return items[0] if items else None # Generic - 自定义泛型 class Stack(Generic[T]): def __init__(self) -> None: self._items: List[T] = [] def push(self, item: T) -> None: self._items.append(item) def pop(self) -> Optional[T]: return self._items.pop() if self._items else None # Protocol - 结构子类型 class Drawable(Protocol): def draw(self) -> None: ... def render(item: Drawable) -> None: item.draw() # 任何有draw方法的类都符合Drawable class Circle: def draw(self) -> None: ... render(Circle()) # OK # Union - 联合类型 def parse(value: Union[str, int]) -> int: if isinstance(value, str): return int(value) return value # Callable - 函数类型 def apply(f: Callable[[int, int], int], x: int, y: int) -> int: return f(x, y) # Optional = Union[X, None] def find(items: List[T], target: T) -> Optional[int]: ...
