struct和class的区别

C/C++ 简单
标签：struct, class, 访问控制

📑 题目

C++中struct和class有什么区别？

💡 参考答案

主要区别：
1. 默认访问权限：struct是public，class是private
2. 默认继承方式：struct是public继承，class是private继承

其他方面完全相同：
- 都可以有成员变量、成员函数
- 都可以有构造函数、析构函数
- 都可以继承、多态
- 都可以模板化

使用习惯：
- struct：主要用于POD（纯数据）类型
- class：用于封装、面向对象设计

💻 代码示例

struct Point { int x, y; // 默认public }; class Point2 { int x, y; // 默认private public: void set(int a, int b) { x = a; y = b; } }; // 继承 struct Derived : Base { }; // public继承 class Derived2 : Base { }; // private继承
