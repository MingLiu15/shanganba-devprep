C++20新特性

C/C++ 困难
标签：C++20, Concepts, 协程, Modules, Ranges

📑 题目

C++20引入了哪些重要新特性？

💡 参考答案

1. Concepts（概念）：
- 对模板参数进行约束
- 提供更清晰的错误信息
- 替代SFINAE

2. Ranges（范围库）：
- 提供函数式风格的数据处理管道
- 惰性求值，性能优化

3. Coroutines（协程）：
- 支持异步编程
- co_await、co_yield、co_return
- 轻量级非抢占式并发

4. Modules（模块）：
- 替代头文件包含
- 编译更快，避免宏污染
- export module / import

5. 其他重要特性：
- 三路比较运算符<=>（太空船运算符）
- std::format（格式化输出）
- std::span（视图类型）
- 指定初始化器
- constexpr扩展（支持更多场景）

💻 代码示例

#include <concepts> #include <ranges> #include <format> // Concepts - 约束模板参数 template<std::integral T> T add(T a, T b) { return a + b; } // 自定义Concept template<typename T> concept Sortable = requires(T& t) { { t.size() } -> std::integral; { t.begin() } -> std::random_access_iterator; }; // Ranges - 函数式管道 auto result = std::views::iota(1, 100) | std::views::filter([](int x) { return x % 2 == 0; }) | std::views::transform([](int x) { return x * x; }) | std::views::take(5); // 三路比较 struct Point { int x, y; auto operator<=>(const Point&) const = default; }; // std::format std::string s = std::format("Hello, {}! Age: {}", "Alice", 25);
