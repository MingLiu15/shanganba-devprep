内联函数的作用和适用场景

C/C++ 简单
标签：inline, 内联函数, 性能优化

📑 题目

什么是内联函数？有什么作用和注意事项？

💡 参考答案

内联函数：建议编译器将函数调用替换为函数体代码，消除调用开销。

作用：
1. 减少函数调用开销（参数压栈、跳转、返回）
2. 保持代码封装性，同时获得宏的性能

适用场景：
- 函数体小（通常<10行）
- 频繁调用
- 简单 getter/setter

注意事项：
1. inline只是建议，编译器可能拒绝
2. 定义通常放在头文件
3. 递归、虚函数、包含循环/复杂控制流通常不内联

💻 代码示例

// 头文件中定义 inline int max(int a, int b) { return a > b ? a : b; } // 类内定义自动内联 class Rectangle { int width, height; public: int getArea() const { // 隐式内联 return width * height; } };
