SQLAlchemy ORM详解

Python 中等
标签：SQLAlchemy, ORM, 数据库, 会话, 关系映射

📑 题目

SQLAlchemy ORM的核心概念有哪些？如何定义模型和关系？

💡 参考答案

**核心组件：**
1. Engine：数据库引擎，管理连接池
2. Session：会话，管理对象状态
3. Base：声明基类，模型继承
4. Column：字段定义
5. relationship：关系定义

**关系类型：**
- 一对多：ForeignKey + relationship
- 多对多：association_proxy 或中间表
- 一对一：uselist=False

**会话操作：**
- add()：添加对象
- commit()：提交事务
- rollback()：回滚
- query()：查询

**查询方法：**
- filter()：过滤条件
- join()：连接查询
- order_by()：排序
- limit()/offset()：分页

💻 代码示例

from sqlalchemy import create_engine, Column, Integer, String, ForeignKey from sqlalchemy.ext.declarative import declarative_base from sqlalchemy.orm import sessionmaker, relationship Base = declarative_base() # 定义模型 class Department(Base): __tablename__ = 'departments' id = Column(Integer, primary_key=True) name = Column(String(100)) employees = relationship('Employee', back_populates='department') class Employee(Base): __tablename__ = 'employees' id = Column(Integer, primary_key=True) name = Column(String(100)) dept_id = Column(Integer, ForeignKey('departments.id')) department = relationship('Department', back_populates='employees') # 创建引擎和会话 engine = create_engine('sqlite:///company.db') Session = sessionmaker(bind=engine) # 创建表 Base.metadata.create_all(engine) # 使用会话 session = Session() # 创建 dept = Department(name='Engineering') session.add(dept) session.commit() emp = Employee(name='Alice', dept_id=dept.id) session.add(emp) session.commit() # 查询 employees = session.query(Employee).filter(Employee.name.like('A%')).all() # 连接查询 results = session.query(Employee, Department).join(Department).all() # 关系访问 for emp in dept.employees: print(emp.name) session.close()
