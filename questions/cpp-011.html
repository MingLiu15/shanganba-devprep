多态的实现方式

C/C++ 中等
标签：多态, 虚函数, 模板, 重载

📑 题目

C++中多态有哪些实现方式？

💡 参考答案

1. 编译时多态（静态多态）：
- 函数重载：同名函数不同参数
- 运算符重载：自定义运算符行为
- 模板：泛型编程

2. 运行时多态（动态多态）：
- 虚函数：通过基类指针/引用调用派生类实现
- 纯虚函数/抽象类：定义接口

区别：静态多态编译期确定，动态多态运行期确定

💻 代码示例

// 静态多态 - 函数重载 void print(int x) { cout << x; } void print(const char* s) { cout << s; } // 静态多态 - 模板 template<typename T> void swap(T& a, T& b) { T t = a; a = b; b = t; } // 动态多态 - 虚函数 class Shape { public: virtual void draw() = 0; // 纯虚函数 }; class Circle : public Shape { void draw() override { /*...*/ } };
