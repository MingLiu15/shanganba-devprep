Python异步编程：asyncio深入

Python 困难
标签：asyncio, 异步编程, 协程, await, 事件循环

📑 题目

asyncio的工作原理是什么？async/await如何使用？

💡 参考答案

**asyncio核心概念：**
- 协程（Coroutine）：使用async def定义的函数
- 事件循环（Event Loop）：调度协程执行
- await：挂起协程，等待结果
- Task：包装协程，用于并发执行

**关键API：**
- asyncio.run()：运行主协程
- asyncio.create_task()：创建任务
- asyncio.gather()：并发执行多个协程
- asyncio.sleep()：非阻塞等待

**并发vs并行：**
- asyncio是单线程并发（协程切换）
- 适合IO密集型任务
- CPU密集型需使用多进程

💻 代码示例

import asyncio async def fetch_data(url): print(f"Fetching {url}") await asyncio.sleep(1) # 模拟IO return f"Data from {url}" async def main(): # 顺序执行 result1 = await fetch_data("url1") result2 = await fetch_data("url2") # 并发执行 tasks = [ asyncio.create_task(fetch_data(f"url{i}")) for i in range(3) ] results = await asyncio.gather(*tasks) print(results) # 运行 asyncio.run(main()) # 与同步代码结合 async def run_in_thread(func, *args): loop = asyncio.get_event_loop() return await loop.run_in_executor(None, func, *args)
