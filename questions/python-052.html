Python数据科学基础：NumPy与Pandas

Python 中等
标签：NumPy, Pandas, 数据分析, DataFrame, 数据科学

📑 题目

NumPy和Pandas的核心概念和常用操作有哪些？

💡 参考答案

**NumPy核心：**
1. ndarray：N维数组，同构数据
2. 向量化运算：避免Python循环，底层C实现
3. 广播机制：不同形状数组运算
4. 切片和索引：支持高级索引

**Pandas核心：**
1. Series：一维带标签数组
2. DataFrame：二维表格数据
3. 索引：loc（标签）、iloc（位置）
4. 分组聚合：groupby + agg
5. 合并：merge、join、concat
6. 时间序列：datetime处理

**性能优化：**
- 向量化替代循环
- 使用category类型减少内存
- 避免逐行操作（iterrows）
- 使用eval/query加速表达式

💻 代码示例
