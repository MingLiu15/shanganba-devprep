BDD行为驱动测试：behave与pytest-bdd

自动化测试 中等
标签：BDD, behave, pytest-bdd, Gherkin, 行为驱动

📑 题目

什么是BDD测试？如何使用behave和pytest-bdd？

💡 参考答案

**BDD（行为驱动开发）：**
- 用自然语言描述系统行为
- 促进业务、开发、测试协作
- 使用Gherkin语法（Given-When-Then）

**Gherkin关键字：**
- Feature：功能描述
- Scenario：测试场景
- Given：前置条件
- When：操作
- Then：预期结果
- And/But：连接步骤

**behave：**
- Python原生BDD框架
- 使用.feature文件描述场景
- 使用Python实现步骤定义

**pytest-bdd：**
- pytest的BDD插件
- 与pytest生态集成
- 支持fixture和参数化

# behave使用示例 # features/login.feature feature_content = """ Feature: 用户登录 Scenario: 成功登录 Given 用户已注册账号 "testuser" 密码 "password123" When 用户使用 "testuser" 和 "password123" 登录 Then 登录成功 """ # features/steps/login_steps.py from behave import given, when, then class UserService: def __init__(self): self.users = {} def register(self, username, password): self.users[username] = password def login(self, username, password): if username in self.users and self.users[username] == password: return {"success": True} return {"success": False} @given("用户已注册账号 {username} 密码 {password}") def step_register(context, username, password): if not hasattr(context, "service"): context.service = UserService() context.service.register(username, password) @when("用户使用 {username} 和 {password} 登录") def step_login(context, username, password): context.result = context.service.login(username, password) @then("登录成功") def step_success(context): assert context.result["success"] # pytest-bdd使用示例 from pytest_bdd import scenario @scenario("login.feature", "成功登录") def test_login(): pass
