Python异常处理深入

Python 中等
标签：异常处理, try-except, 自定义异常, 异常链

📑 题目

Python异常处理的完整语法是什么？如何自定义异常？

💡 参考答案

**异常处理语法：**
- try/except/else/finally
- except可以捕获多个异常
- as获取异常对象
- raise抛出异常

**自定义异常：**
- 继承Exception或其子类
- 可以添加自定义属性

**异常链：**
- raise ... from ...：保留原始异常信息
- __cause__属性

**最佳实践：**
- 捕获具体异常而非Exception
- 不要捕获所有异常后pass
- 使用finally或上下文管理器确保资源释放

💻 代码示例

class ValidationError(Exception): def __init__(self, field, message): self.field = field self.message = message super().__init__(f"{field}: {message}") def validate_age(age): try: age = int(age) if age < 0 or age > 150: raise ValidationError("age", "Invalid age range") return age except ValueError as e: raise ValidationError("age", "Must be integer") from e # 异常链 try: validate_age("abc") except ValidationError as e: print(f"Error: {e}") print(f"Cause: {e.__cause__}") # 完整语法 try: result = 1 / 1 except ZeroDivisionError: print("Division by zero") else: print(f"Result: {result}") # 无异常时执行 finally: print("Cleanup") # 总是执行
