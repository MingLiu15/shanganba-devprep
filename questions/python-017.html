Python内存管理

Python 中等
标签：进阶特性, 内存管理, 垃圾回收

📑 题目

介绍Python的内存管理机制，包括内存分配、对象池和缓存机制。

💡 参考答案

Python的内存管理分为多个层次：

1. 内存分配器层次：
- PyObject_Malloc：Python自己的内存分配器，针对小对象优化。
- pymalloc：管理小于512字节的小对象内存池，减少系统malloc调用。
- 大于512字节的对象直接调用系统malloc/free。

2. 对象缓存机制：
- 小整数缓存：-5到256的整数对象被预创建并缓存。
- 字符串intern：短字符串和标识符会被intern，相同内容的字符串共享对象。
- 空元组缓存：()被全局缓存复用。

3. 内存优化技巧：
- 使用__slots__减少实例的__dict__开销。
- 使用array模块或numpy代替大列表。
- 使用生成器代替列表处理大数据。
- 使用__slots__、弱引用（weakref）等减少内存占用。

4. 内存分析工具：
- sys.getsizeof()：获取对象大小。
- gc.get_objects()：获取所有被GC管理的对象。
- tracemalloc：跟踪内存分配。
- 第三方工具：memory_profiler、objgraph、pympler。

5. 内存泄漏排查：
- 循环引用 + __del__方法可能导致内存泄漏。
- 全局变量持有大对象引用。
- 使用weakref打破循环引用。

💻 代码示例

import sys import weakref # 对象大小 print(sys.getsizeof(1)) # 28 print(sys.getsizeof('hello')) # 54 print(sys.getsizeof([])) # 56 print(sys.getsizeof({})) # 64 # 小整数缓存 a = 100 b = 100 print(a is b) # True # __slots__ 减少内存 class Normal: def __init__(self, x, y): self.x = x self.y = y class Slotted: __slots__ = ('x', 'y') def __init__(self, x, y): self.x = x self.y = y print(sys.getsizeof(Normal(1,2))) # 56 + __dict__ print(sys.getsizeof(Slotted(1,2))) # 48（更小） # 弱引用 class Big: pass obj = Big() ref = weakref.ref(obj) print(ref() is obj) # True del obj print(ref()) # None（对象已被回收）
