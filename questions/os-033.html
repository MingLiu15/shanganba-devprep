生产者-消费者问题

操作系统 中等
标签：生产者消费者, 同步, 信号量, 经典问题, 并发

📑 题目

什么是生产者-消费者问题？如何用信号量实现？

💡 参考答案

问题描述：
- 生产者向缓冲区放入产品
- 消费者从缓冲区取出产品
- 缓冲区满时生产者等待，缓冲区空时消费者等待

需要三个信号量：
1. mutex：互斥访问缓冲区（初始值1）
2. empty：空槽位数量（初始值N）
3. full：已有产品数量（初始值0）

生产者流程：
1. P(empty)：等待空槽位
2. P(mutex)：获取互斥锁
3. 放入产品
4. V(mutex)：释放互斥锁
5. V(full)：通知消费者

消费者流程：
1. P(full)：等待产品
2. P(mutex)：获取互斥锁
3. 取出产品
4. V(mutex)：释放互斥锁
5. V(empty)：通知生产者

注意：P(empty)和P(mutex)的顺序不能颠倒，否则可能死锁。

💻 代码示例

#include <semaphore.h> #include <pthread.h> #define N 10 int buffer[N]; int in = 0, out = 0; sem_t empty, full, mutex; // 初始化 sem_init(&empty, 0, N); // N个空槽 sem_init(&full, 0, 0); // 0个产品 sem_init(&mutex, 0, 1); // 互斥 void* producer(void* arg) { int item; while (true) { item = produce(); // 生产 sem_wait(&empty); // 等待空槽 sem_wait(&mutex); // 加锁 buffer[in] = item; in = (in + 1) % N; sem_post(&mutex); // 解锁 sem_post(&full); // 通知消费者 } } void* consumer(void* arg) { int item; while (true) { sem_wait(&full); // 等待产品 sem_wait(&mutex); // 加锁 item = buffer[out]; out = (out + 1) % N; sem_post(&mutex); // 解锁 sem_post(&empty); // 通知生产者 consume(item); // 消费 } }
