pathlib模块

Python 简单
标签：pathlib, 路径操作, 文件系统, 标准库, 现代Python

📑 题目

pathlib模块与os.path相比有什么优势？

💡 参考答案

pathlib：面向对象的文件路径操作模块（Python 3.4+），替代os.path。

优势：
1. 面向对象：Path对象而非字符串
2. 跨平台：自动处理Windows/Unix路径差异
3. 操作符重载：/ 连接路径
4. 链式调用：方法返回新Path对象
5. 更丰富的方法：读写、权限、遍历等

常用操作：
- Path('dir') / 'file.txt'：连接路径
- exists()：是否存在
- is_file() / is_dir()：类型判断
- mkdir() / rmdir()：创建/删除目录
- read_text() / write_text()：读写文本
- glob()：模式匹配
- iterdir()：遍历目录

💻 代码示例

from pathlib import Path # 创建路径 p = Path('/home/user') / 'documents' / 'file.txt' # 等价于 os.path.join('/home/user', 'documents', 'file.txt') # 路径属性 print(p.name) # file.txt print(p.stem) # file print(p.suffix) # .txt print(p.parent) # /home/user/documents print(p.parts) # ('/', 'home', 'user', 'documents', 'file.txt') # 文件操作 p.exists() p.is_file() p.is_dir() # 读写（Python 3.5+） Path('config.txt').write_text('setting=value') content = Path('config.txt').read_text() # 遍历目录 for f in Path('.').iterdir(): if f.is_file(): print(f) # glob模式匹配 list(Path('.').glob('*.py')) # 当前目录所有py文件 list(Path('.').rglob('*.py')) # 递归所有py文件 # 创建目录 Path('new/dir').mkdir(parents=True, exist_ok=True) # 相对/绝对路径 Path('file.txt').resolve() # 转为绝对路径 Path('/a/b').relative_to('/a') # 计算相对路径
