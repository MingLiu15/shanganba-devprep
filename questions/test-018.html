Allure测试报告

自动化测试 中等
标签：Allure, 测试报告, 可视化, pytest

📑 题目

如何使用Allure生成测试报告？

💡 参考答案

**Allure特点：**
1. 美观的Web报告界面
2. 支持历史趋势分析
3. 支持用例分类和过滤
4. 支持附件（截图、日志）
5. 支持步骤和严重级别

**安装：**
```bash
pip install allure-pytest
```

**常用装饰器：**
- `@allure.feature`：功能模块
- `@allure.story`：用户故事
- `@allure.severity`：严重级别
- `@allure.step`：测试步骤
- `@allure.attach`：添加附件

💻 代码示例

import allure import pytest @allure.feature("用户管理") class TestUser: @allure.story("用户登录") @allure.severity(allure.severity_level.CRITICAL) def test_login_success(self): """测试正常登录""" with allure.step("输入用户名"): username = "admin" with allure.step("输入密码"): password = "password123" with allure.step("点击登录"): result = login(username, password) with allure.step("验证登录结果"): assert result["status"] == "success" allure.attach(str(result), "登录响应", allure.attachment_type.JSON) @allure.story("用户注册") @allure.severity(allure.severity_level.NORMAL) def test_register(self): """测试用户注册""" user_data = {"username": "newuser", "email": "new@example.com"} allure.attach.dict(user_data, "注册数据") result = register(user_data) assert result["success"] # ===== 运行和生成报告 ===== # 运行测试并收集数据 # pytest --alluredir=./allure-results # 生成报告 # allure generate ./allure-results -o ./allure-report # 打开报告 # allure open ./allure-report # 清除历史数据 # allure serve ./allure-results
