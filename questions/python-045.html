Flask框架核心概念

Python 中等
标签：Flask, Web框架, 路由, 蓝图, 中间件

📑 题目

Flask框架的核心概念有哪些？路由、蓝图、中间件如何使用？

💡 参考答案

**Flask核心概念：**

1. **应用对象**：Flask(__name__) 创建应用实例
2. **路由**：@app.route() 装饰器绑定URL和视图函数
3. **请求对象**：request 包含请求数据
4. **响应对象**：Response 或直接返回字符串/元组
5. **蓝图**：Blueprint 模块化组织应用
6. **上下文**：应用上下文(app context)和请求上下文(request context)

**蓝图使用：**
```python
from flask import Blueprint
user_bp = Blueprint('user', __name__)

@user_bp.route('/profile')
def profile():
return 'User Profile'

# 注册蓝图
app.register_blueprint(user_bp, url_prefix='/user')
```

**中间件模式：**
- before_request：请求前处理
- after_request：请求后处理
- teardown_request：请求结束清理

💻 代码示例

from flask import Flask, request, Blueprint, jsonify app = Flask(__name__) # 基本路由 @app.route('/') def index(): return 'Hello Flask' # 动态路由 @app.route('/user/<int:user_id>') def get_user(user_id): return f'User {user_id}' # HTTP方法 @app.route('/api/data', methods=['GET', 'POST']) def api_data(): if request.method == 'POST': data = request.get_json() return jsonify({'received': data}) return jsonify({'method': 'GET'}) # 蓝图 api_bp = Blueprint('api', __name__) @api_bp.route('/users') def list_users(): return jsonify(['user1', 'user2']) app.register_blueprint(api_bp, url_prefix='/api/v1') # 中间件 @app.before_request def before_request(): print(f'请求: {request.path}') @app.after_request def after_request(response): response.headers['X-Custom'] = 'value' return response if __name__ == '__main__': app.run(debug=True)
