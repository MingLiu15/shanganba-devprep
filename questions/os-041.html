📑 题目

什么是协程？协程与线程有什么区别？

💡 参考答案

协程（Coroutine）：用户态的轻量级线程，由程序自己调度，而非操作系统。

特点：
1. 协作式调度：主动让出执行权（yield），而非抢占式
2. 切换开销小：不需要进入内核，只需保存/恢复寄存器
3. 高并发：单线程可运行数万协程
4. 无锁编程：同一协程内无并发问题

与线程对比：
| 特性 | 线程 | 协程 |
|------|------|------|
| 调度 | 操作系统抢占 | 用户协作 |
| 切换开销 | 大（内核态） | 小（用户态） |
| 并发数 | 数千 | 数百万 |
| 数据共享 | 需要同步 | 单线程无锁 |

应用场景：
- 高并发IO（网络服务器）
- 异步编程简化
- 生成器/迭代器

// C++20协程（简化示例） #include <coroutine> // 生成器协程 struct Generator { struct promise_type { int current_value; auto get_return_object() { return Generator{this}; } auto initial_suspend() { return std::suspend_always{}; } auto final_suspend() noexcept { return std::suspend_always{}; } void unhandled_exception() {} void return_void() {} auto yield_value(int value) { current_value = value; return std::suspend_always{}; } }; using Handle = std::coroutine_handle<promise_type>; Handle handle; Generator(promise_type* p) : handle(Handle::from_promise(*p)) {} ~Generator() { if (handle) handle.destroy(); } int next() { handle.resume(); return handle.promise().current_value; } }; Generator fibonacci() { co_yield 1; co_yield 1; int a = 1, b = 1; while (true) { int c = a + b; co_yield c; a = b; b = c; } } // 使用 auto gen = fibonacci(); for (int i = 0; i < 10; i++) { std::cout << gen.next() << " "; }
