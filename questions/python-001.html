可变类型与不可变类型的区别

Python 简单
标签：基础语法, 数据类型, 可变性

📑 题目

请详细说明Python中可变类型与不可变类型的区别，并列举常见的数据类型。

💡 参考答案

Python中数据类型分为可变（mutable）和不可变（immutable）两类。

1. 不可变类型：int、float、str、tuple、frozenset、bytes。一旦创建，值不可修改，修改时会创建新对象。
2. 可变类型：list、dict、set、bytearray。创建后可以原地修改内容，对象地址不变。

关键区别：
- 不可变类型作为字典的key、集合的元素；可变类型不能。
- 函数传参时，不可变类型类似值传递，可变类型类似引用传递，函数内修改可变对象会影响外部。
- 不可变类型更安全，适合多线程环境共享数据；可变类型需注意线程安全。
- tuple虽然不可变，但如果包含可变元素（如list），该元素的值仍可被修改。

💻 代码示例

# 不可变类型示例 a = (1, 2, [3, 4]) # a[0] = 10 # TypeError: 'tuple' object does not support item assignment a[2][0] = 10 # 但内部可变元素可以修改 print(a) # (1, 2, [10, 4]) # 可变类型示例 b = [1, 2, 3] print(id(b)) # 140xxx b.append(4) print(id(b)) # 140xxx，地址不变 # 函数传参的影响 def modify(lst): lst.append(99) my_list = [1, 2] modify(my_list) print(my_list) # [1, 2, 99]
