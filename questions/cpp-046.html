预处理指令

C/C++ 简单
标签：预处理, #define, #ifdef, 条件编译, 头文件守卫

📑 题目

列举C/C++常用的预处理指令及其作用。

💡 参考答案

预处理在编译前执行，处理以#开头的指令。

常用指令：
1. #include：包含头文件
2. #define：定义宏
3. #ifdef/#ifndef/#endif：条件编译
4. #if/#elif/#else：条件判断
5. #pragma：编译器特定指令

条件编译用途：
- 跨平台代码：#ifdef _WIN32
- 调试开关：#ifdef DEBUG
- 头文件守卫：防止重复包含

预定义宏：
- __FILE__：当前文件名
- __LINE__：当前行号
- __DATE__：编译日期
- __TIME__：编译时间
- __cplusplus：C++标准版本

注意：
- 预处理不做语法检查
- 宏只是文本替换
- 优先使用const/enum/inline替代宏

💻 代码示例

// 头文件守卫（传统方式） #ifndef MYHEADER_H #define MYHEADER_H // 头文件内容 #endif // 现代方式（C++17） #pragma once // 条件编译 #ifdef _WIN32 #include <windows.h> #elif defined(__linux__) #include <unistd.h> #endif // 调试宏 #ifdef DEBUG #define LOG(msg) printf("[DEBUG] %s\n", msg) #else #define LOG(msg) #endif // 预定义宏 printf("file: %s, line: %d\n", __FILE__, __LINE__);
