四种类型转换运算符

C/C++ 中等
标签：类型转换, static_cast, dynamic_cast, const_cast, reinterpret_cast

📑 题目

说明C++四种类型转换运算符的区别和使用场景。

💡 参考答案

1. static_cast：编译期静态转换，用于相关类型（如int↔double，基类↔派生类），无运行时检查
2. dynamic_cast：运行时动态转换，用于多态类型（必须有虚函数），失败返回nullptr（指针）或抛异常（引用）
3. const_cast：添加或移除const属性，用于调用非const接口
4. reinterpret_cast：底层二进制重新解释，最危险，用于指针/引用↔指针/引用

💻 代码示例

double d = 3.14; int i = static_cast<int>(d); Base* b = new Derived(); Derived* d = dynamic_cast<Derived*>(b); // 安全向下转型 const int* p = &i; int* q = const_cast<int*>(p); // 去const intptr_t addr = reinterpret_cast<intptr_t>(p);
