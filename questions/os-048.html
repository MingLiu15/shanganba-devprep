Linux系统编程：fork/exec/wait

操作系统 困难
标签：Linux, 系统编程, fork, exec, wait, 进程控制

📑 题目

fork、exec和wait系统调用的作用是什么？如何使用？

💡 参考答案

**fork()：**
- 创建子进程，复制父进程地址空间
- 父子进程从fork返回处继续执行
- 返回值：父进程返回子进程PID，子进程返回0
- 写时复制（COW）优化性能

**exec()系列：**
- 用新程序替换当前进程映像
- 不创建新进程，只是替换
- 成功不返回，失败返回-1

**wait()/waitpid()：**
- 父进程等待子进程结束
- 获取子进程退出状态
- 防止僵尸进程

**典型流程：**
fork -> 子进程exec -> 父进程wait

💻 代码示例

#include <unistd.h> #include <sys/wait.h> #include <stdio.h> int main() { pid_t pid = fork(); if (pid < 0) { perror("fork failed"); return 1; } else if (pid == 0) { // 子进程 printf("Child: PID=%d, PPID=%d\n", getpid(), getppid()); // 执行新程序 execlp("ls", "ls", "-l", NULL); // execlp成功不返回，到这里说明失败 perror("execlp failed"); return 1; } else { // 父进程 printf("Parent: PID=%d, Child PID=%d\n", getpid(), pid); int status; pid_t child_pid = wait(&status); // 等待子进程结束 if (WIFEXITED(status)) { printf("Child %d exited with code %d\n", child_pid, WEXITSTATUS(status)); } } return 0; }
