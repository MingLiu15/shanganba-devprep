itertools模块

Python 中等
标签：itertools, 迭代器, 标准库, 函数式编程

📑 题目

介绍itertools模块的常用函数。

💡 参考答案

itertools：高效迭代器工具模块，用于创建和操作迭代器。

常用函数：
1. 无限迭代器：
- count(start, step)：无限计数
- cycle(iterable)：无限循环
- repeat(elem, n)：重复n次

2. 组合迭代器：
- product：笛卡尔积
- permutations：排列
- combinations：组合
- combinations_with_replacement：可重复组合

3. 筛选/分组：
- chain：连接多个迭代器
- groupby：分组（需先排序）
- filterfalse：反向过滤
- takewhile/dropwhile：条件取/跳

4. 其他：
- accumulate：累积计算
- zip_longest：zip的变体，填充缺失值

💻 代码示例

import itertools # 无限迭代器（需配合break使用） for i in itertools.count(10, 2): # 从10开始，步长2 if i > 20: break print(i) # 10, 12, 14, 16, 18, 20 # 组合 list(itertools.permutations([1,2,3], 2)) # [(1,2), (1,3), (2,1), (2,3), (3,1), (3,2)] list(itertools.combinations([1,2,3,4], 2)) # [(1,2), (1,3), (1,4), (2,3), (2,4), (3,4)] # 笛卡尔积 list(itertools.product('AB', '12')) # [('A','1'), ('A','2'), ('B','1'), ('B','2')] # 累积 list(itertools.accumulate([1,2,3,4,5])) # [1, 3, 6, 10, 15] # 分组 data = [('A', 1), ('A', 2), ('B', 3), ('B', 4)] for key, group in itertools.groupby(data, key=lambda x: x[0]): print(key, list(group)) # A [(A,1), (A,2)] # B [(B,3), (B,4)] # 连接迭代器 list(itertools.chain([1,2], [3,4], [5,6])) # [1,2,3,4,5,6]
