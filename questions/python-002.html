深拷贝与浅拷贝的区别

Python 简单
标签：基础语法, 拷贝, copy模块

📑 题目

解释Python中浅拷贝（shallow copy）和深拷贝（deep copy）的区别，并说明各自的实现方式。

💡 参考答案

浅拷贝：创建新对象，但只复制顶层引用。如果原对象包含嵌套的可变对象，浅拷贝和原对象共享这些嵌套对象。实现方式：copy.copy()、list()、dict()、切片[:]、list.copy()。

深拷贝：递归复制所有层级的对象，完全独立于原对象。修改深拷贝不会影响原对象。实现方式：copy.deepcopy()。

区别总结：
- 浅拷贝只复制第一层，嵌套对象仍共享引用。
- 深拷贝递归复制所有层级，完全独立。
- 深拷贝性能开销更大，尤其是对象层级很深或包含大量数据时。
- 对于只包含不可变类型的对象，浅拷贝和深拷贝效果相同。
- 对象的__copy__和__deepcopy__方法可以自定义拷贝行为。

💻 代码示例

import copy # 浅拷贝 a = [1, [2, 3], 4] b = a[:] # 浅拷贝 b[1][0] = 99 print(a) # [1, [99, 3], 4] 被影响 # 深拷贝 c = [1, [2, 3], 4] d = copy.deepcopy(c) # 深拷贝 d[1][0] = 99 print(c) # [1, [2, 3], 4] 不受影响 # copy模块 import copy lst = [[1, 2], [3, 4]] shallow = copy.copy(lst) deep = copy.deepcopy(lst) shallow[0][0] = 99 print(lst) # [[99, 2], [3, 4]] 浅拷贝影响原对象
