二叉树的遍历方式

C/C++ 中等
标签：二叉树, 遍历, 前序, 中序, 后序, 层序, 递归, 算法

📑 题目

列举二叉树的各种遍历方式及其实现。

💡 参考答案

四种遍历方式：

1. 前序遍历（根-左-右）：
- 先访问根节点，再左子树，再右子树
- 应用：复制树、序列化

2. 中序遍历（左-根-右）：
- 先左子树，再根节点，再右子树
- BST中序遍历得到有序序列

3. 后序遍历（左-右-根）：
- 先左子树，再右子树，再根节点
- 应用：释放树、计算目录大小

4. 层序遍历（BFS）：
- 按层从上到下、从左到右
- 使用队列实现
- 应用：求树的宽度、最短路径

实现方式：
- 递归：代码简洁
- 迭代（栈）：避免递归栈溢出
- Morris遍历：O(1)空间，修改树结构

💻 代码示例

struct TreeNode { int val; TreeNode *left, *right; }; // 前序遍历（递归） void preorder(TreeNode* root) { if (!root) return; cout << root->val << " "; preorder(root->left); preorder(root->right); } // 中序遍历（迭代） void inorder(TreeNode* root) { stack<TreeNode*> stk; TreeNode* cur = root; while (cur || !stk.empty()) { while (cur) { stk.push(cur); cur = cur->left; } cur = stk.top(); stk.pop(); cout << cur->val << " "; cur = cur->right; } } // 层序遍历（BFS） void levelOrder(TreeNode* root) { queue<TreeNode*> q; if (root) q.push(root); while (!q.empty()) { int size = q.size(); for (int i = 0; i < size; i++) { TreeNode* node = q.front(); q.pop(); cout << node->val << " "; if (node->left) q.push(node->left); if (node->right) q.push(node->right); } } }
