属性语法attribute

C/C++ 中等
标签：attribute, C++11, nodiscard, deprecated, 编译器指令

📑 题目

C++11中的属性语法(attribute)是什么？有哪些常用属性？

💡 参考答案

属性语法：为编译器提供额外信息的标注方式。

语法：[[attribute]] 或 [[namespace::attribute]]

常用属性：
1. [[nodiscard]]：返回值不应被忽略
2. [[deprecated]]：标记废弃接口
3. [[maybe_unused]]：变量可能未使用，避免警告
4. [[fallthrough]]：switch中故意fallthrough
5. [[noreturn]]：函数不返回

编译器特定属性：
- GCC：__attribute__((...))
- MSVC：__declspec(...)
- C++11统一为[[...]]语法

应用场景：
- API设计：标记废弃函数
- 代码质量：nodiscard防止忽略错误码
- 消除警告：maybe_unused

💻 代码示例

// nodiscard：返回值不应忽略 [[nodiscard]] int allocateResource() { return getResourceId(); } // 如果调用时忽略返回值，编译器警告 // allocateResource(); // 警告！ int id = allocateResource(); // OK // deprecated：标记废弃 [[deprecated("Use newFunction() instead")]] void oldFunction() {} // maybe_unused：可能未使用 void func([[maybe_unused]] int param) { // 如果param未使用，不报警告 } // fallthrough：故意fallthrough switch (x) { case 1: doSomething(); [[fallthrough]]; // 告诉编译器这是故意的 case 2: doOther(); break; } // noreturn：函数不返回 [[noreturn]] void terminate() { throw std::runtime_error("Terminated"); }
