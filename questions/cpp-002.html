new和malloc的区别

C/C++ 简单
标签：内存管理, new, malloc

📑 题目

比较C++的new操作符和C的malloc函数的区别。

💡 参考答案

1. 类型：new是运算符，malloc是函数
2. 类型安全：new自动计算类型大小，malloc需要手动指定字节数
3. 初始化：new可以调用构造函数初始化，malloc不初始化
4. 返回值：new返回正确类型指针，malloc返回void*需要强制转换
5. 失败处理：new抛出bad_alloc异常，malloc返回NULL
6. 释放：new用delete，malloc用free，不能混用

💻 代码示例

// new方式 int* p1 = new int(5); // 初始化值为5 int* arr = new int[10]; // 数组 delete p1; delete[] arr; // malloc方式 int* p2 = (int*)malloc(sizeof(int)); *p2 = 5; free(p2);
