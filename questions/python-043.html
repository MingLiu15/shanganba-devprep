正则表达式re模块

Python 中等
标签：正则表达式, regex, re, 字符串匹配, 模式匹配

📑 题目

介绍Python的re模块常用功能。

💡 参考答案

re模块：正则表达式支持，用于字符串模式匹配。

常用函数：
- match：从字符串开头匹配
- search：搜索第一个匹配
- findall：查找所有匹配
- finditer：返回匹配迭代器
- sub：替换匹配
- split：按模式分割
- compile：编译模式（重复使用）

常用模式：
- .：任意字符
- *：0次或多次
- +：1次或多次
- ?：0次或1次
- {n,m}：n到m次
- []：字符集
- ()：分组
- ^ $：开始/结束

💻 代码示例

import re text = 'Email: alice@example.com, Phone: 138-1234-5678' # 搜索 match = re.search(r'\w+@\w+\.\w+', text) if match: print(match.group()) # alice@example.com # 查找所有 emails = re.findall(r'\w+@\w+\.\w+', text) # 分组 match = re.search(r'(\w+)@((\w+)\.(\w+))', text) if match: print(match.group(0)) # 完整匹配 print(match.group(1)) # 用户名 print(match.group(2)) # 域名 # 替换 masked = re.sub(r'\d{3}-\d{4}-\d{4}', '***-****-****', text) # 分割 parts = re.split(r'[,;\s]+', 'a,b; c d') # 编译（重复使用） pattern = re.compile(r'\d+') pattern.findall('abc123def456') # ['123', '456'] # 命名分组 match = re.search(r'(?P<user>\w+)@(?P<domain>\w+\.\w+)', text) print(match.group('user')) # 贪婪vs非贪婪 re.findall(r'<.*>', '<a><b>') # ['<a><b>'] 贪婪 re.findall(r'<.*?>', '<a><b>') # ['<a>', '<b>'] 非贪婪
