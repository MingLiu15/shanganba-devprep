最小生成树：Kruskal与Prim算法

算法题 中等
标签：最小生成树, Kruskal, Prim, 并查集, 贪心, 图论

📑 题目

什么是最小生成树？Kruskal和Prim算法的区别？

💡 参考答案

**最小生成树（MST）：**
给定带权无向连通图，找到一棵包含所有顶点的树，使得边权之和最小。

**Kruskal算法：**
1. 将所有边按权重排序
2. 依次选取最小边，若不形成环则加入
3. 使用并查集判断是否形成环
4. 时间复杂度：O(E log E)

**Prim算法：**
1. 从任意顶点开始
2. 每次选择连接已选集和未选集的最小边
3. 使用优先队列（最小堆）优化
4. 时间复杂度：O(E log V)（堆优化版）

**选择建议：**
- 稀疏图（E << V^2）：用Kruskal
- 稠密图（E ~ V^2）：用Prim（邻接矩阵版O(V^2)）

**应用场景：**
- 网络布线、道路规划
- 聚类分析
- 近似算法（如TSP的MST近似）

💻 代码示例

from typing import List, Tuple import heapq # ===== Kruskal算法（并查集）===== class UnionFind: def __init__(self, n): self.parent = list(range(n)) self.rank = [0] * n def find(self, x): if self.parent[x] != x: self.parent[x] = self.find(self.parent[x]) # 路径压缩 return self.parent[x] def union(self, x, y): px, py = self.find(x), self.find(y) if px == py: return False if self.rank[px] < self.rank[py]: px, py = py, px self.parent[py] = px if self.rank[px] == self.rank[py]: self.rank[px] += 1 return True def kruskal(n: int, edges: List[Tuple[int, int, int]]) -> int: """Kruskal最小生成树 Args: n: 节点数 edges: [(weight, u, v), ...] Returns: 最小生成树权重和 """ edges.sort() # 按权重排序 uf = UnionFind(n) total_weight = 0 edge_count = 0 for w, u, v in edges: if uf.union(u, v): total_weight += w edge_count += 1 if edge_count == n - 1: break return total_weight # ===== Prim算法（堆优化）===== def prim(n: int, graph: List[List[Tuple[int, int]]]) -> int: """Prim最小生成树（堆优化） Args: n: 节点数 graph: 邻接表 graph[u] = [(v, weight), ...] Returns: 最小生成树权重和 """ visited = [False] * n min_heap = [(0, 0)] # (weight, node) total_weight = 0 edge_count = 0 while min_heap and edge_count < n: w, u = heapq.heappop(min_heap) if visited[u]: continue visited[u] = True total_weight += w edge_count += 1 for v, weight in graph[u]: if not visited[v]: heapq.heappush(min_heap, (weight, v)) return total_weight # ===== 测试 ===== # 5个节点的图 # edges: (weight, u, v) edges = [ (1, 0, 1), (3, 0, 2), (4, 0, 3), (2, 1, 2), (5, 1, 3), (6, 2, 3), (7, 2, 4), (8, 3, 4) ] print(f"Kruskal MST: {kruskal(5, edges)}") # 1+2+6+8=17 # 构建邻接表 graph = [[] for _ in range(5)] for w, u, v in edges: graph[u].append((v, w)) graph[v].append((u, w)) print(f"Prim MST: {prim(5, graph)}") # 17
