Python中__slots__的作用

Python 中等
标签：高级特性, __slots__, 内存优化, 属性限制

📑 题目

解释Python中__slots__的作用、优缺点及适用场景。

💡 参考答案

__slots__是一个类属性，用于显式声明实例可以拥有的属性名称列表，替代默认的__dict__。

作用：
1. 节省内存：不使用__dict__，每个实例节省约40-50%的内存。
2. 加快属性访问速度：通过描述符直接访问，比字典查找更快。
3. 限制属性：只能赋值__slots__中声明的属性，防止动态添加属性。

工作原理：
- 定义__slots__后，Python为每个slot创建描述符（descriptor），存储在类级别。
- 实例不再有__dict__（除非__slots__中包含'__dict__'）。
- 属性值直接存储在实例的底层结构中，而非字典中。

注意事项：
- 子类不会继承父类的__slots__，除非子类也定义了__slots__。
- 如果父类没有定义__slots__，子类定义__slots__不会节省内存（因为__dict__仍存在）。
- 不能在实例上添加__slots__中未声明的属性。
- __slots__中可以包含'__dict__'和'__weakref__'来保留这些功能。
- 与某些序列化库（如pickle）可能有兼容性问题。

适用场景：
- 创建大量实例时（如ORM模型、游戏对象、数据容器）。
- 属性固定的类。
- 内存敏感的应用。

💻 代码示例

class Normal: def __init__(self, x, y): self.x = x self.y = y class Slotted: __slots__ = ('x', 'y') def __init__(self, x, y): self.x = x self.y = y import sys n = Normal(1, 2) s = Slotted(1, 2) print(f'Normal: {sys.getsizeof(n)} bytes') # 56 print(f'Slotted: {sys.getsizeof(s)} bytes') # 48 # 属性限制 s.x = 10 # OK # s.z = 30 # AttributeError: 'Slotted' object has no attribute 'z' # 批量创建时内存差异明显 import tracemalloc tracemalloc.start() normal_objs = [Normal(i, i) for i in range(100000)] n1, n2 = tracemalloc.get_traced_memory() tracemalloc.stop() tracemalloc.start() slotted_objs = [Slotted(i, i) for i in range(100000)] s1, s2 = tracemalloc.get_traced_memory() tracemalloc.stop() print(f'Normal: {n1/1024:.1f}KB, Slotted: {s1/1024:.1f}KB')
