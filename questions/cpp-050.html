future/promise/async

C/C++ 中等
标签：future, promise, async, 异步, 并发

📑 题目

解释C++11中的future、promise和async的用法。

💡 参考答案

future/promise：异步编程模型，用于线程间传递结果。

async：启动异步任务，返回future对象。
- std::launch::async：在新线程执行
- std::launch::deferred：延迟执行（调用get时执行）

future：表示异步操作的结果，可以获取结果或等待完成。
- get()：获取结果（阻塞等待）
- wait()：等待完成
- wait_for()：限时等待

promise：存储结果，供关联的future获取。
- set_value()：设置结果
- set_exception()：设置异常

使用场景：
1. 异步计算：启动耗时操作，稍后获取结果
2. 线程间通信：一个线程计算，另一个获取结果

💻 代码示例

#include <future> #include <iostream> // async简单用法 int compute() { std::this_thread::sleep_for(std::chrono::seconds(1)); return 42; } std::future<int> fut = std::async(std::launch::async, compute); // 做其他事情... int result = fut.get(); // 阻塞等待结果 // promise/future用法 void producer(std::promise<int>& p) { p.set_value(100); // 设置结果 } void consumer(std::future<int>& f) { int val = f.get(); // 获取结果 } std::promise<int> prom; std::future<int> fut2 = prom.get_future(); std::thread t1(producer, std::ref(prom)); std::thread t2(consumer, std::ref(fut2)); // 处理多个future std::vector<std::future<int>> futures; for (int i = 0; i < 10; i++) { futures.push_back(std::async(compute)); } for (auto& f : futures) { std::cout << f.get() << " "; }
