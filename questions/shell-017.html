正则表达式在Shell中的应用

Shell脚本 困难
标签：正则表达式, grep, sed, awk, 模式匹配

📑 题目

Shell中如何使用正则表达式？

💡 参考答案

**正则表达式类型：**
1. **基本正则（BRE）** - grep默认
2. **扩展正则（ERE）** - grep -E 或 egrep
3. **Perl正则（PCRE）** - grep -P

**常用元字符：**
- `.` - 任意字符
- `^` `$` - 行首行尾
- `*` `+` `?` - 重复次数
- `[]` - 字符集
- `()` - 分组
- `|` - 或
- `\b` - 单词边界
- `\d` - 数字（PCRE）

**使用场景：**
- grep -E/grep -P
- sed -r/sed -E
- awk
- [[ var =~ regex ]]
