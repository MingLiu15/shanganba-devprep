📑 题目

如何使用json-server和wiremock搭建Mock服务器？

💡 参考答案

**Mock服务器作用：**
1. 模拟后端API，前后端并行开发
2. 隔离测试，不依赖真实服务
3. 模拟异常场景（超时、错误）
4. 控制响应数据，便于测试

**json-server：**
- 基于JSON文件快速搭建REST API
- 支持CRUD操作
- 支持过滤、分页、排序
- 适合前端开发和简单测试

**WireMock：**
- 功能强大的HTTP Mock服务器
- 支持请求匹配、响应模板
- 支持延迟、故障注入
- 适合复杂测试场景

💻 代码示例
