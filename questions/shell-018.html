📑 题目

如何调试Shell脚本？有哪些常用技巧？

💡 参考答案

**调试选项：**
- `bash -x script.sh` - 显示执行的每条命令
- `bash -v script.sh` - 显示读取的每行
- `bash -n script.sh` - 语法检查（不执行）

**set命令：**
- `set -x` / `set +x` - 开启/关闭调试
- `set -e` - 命令失败立即退出
- `set -u` - 使用未定义变量报错
- `set -o pipefail` - 管道失败传递
- `set -o nounset` - 同 -u

**常用组合：**
```bash
#!/bin/bash
set -euo pipefail # 严格模式
```

**调试技巧：**
1. 在关键位置添加 `echo` 或 `printf`
2. 使用 `trap` 捕获错误
3. 检查退出状态 `$?`
4. 使用 `tee` 记录执行过程
