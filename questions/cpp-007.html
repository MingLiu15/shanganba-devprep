const关键字的用法

C/C++ 中等
标签：const, 常量, 类型安全

📑 题目

列举C++中const关键字的各种用法。

💡 参考答案

1. const变量：值不可修改
2. const指针：
- const int* p：指向常量的指针（内容不能改）
- int* const p：常量指针（指向不能改）
- const int* const p：指向常量的常量指针
3. const引用：函数参数，避免拷贝且保证不修改
4. const成员函数：不修改对象状态，可被const对象调用
5. constexpr：编译期常量

💻 代码示例

const int MAX = 100; // 常量 void func(const string& s); // const引用参数 class MyClass { public: int getValue() const; // const成员函数 }; constexpr int square(int x) { // 编译期计算 return x * x; }
