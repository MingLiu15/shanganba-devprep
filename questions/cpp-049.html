placement new

C/C++ 困难
标签：placement new, 内存管理, 定位new, 对象池

📑 题目

什么是placement new？它有什么应用场景？

💡 参考答案

placement new：在已分配的内存上构造对象，不分配新内存。

语法：new (address) Type(args);

应用场景：
1. 内存池：预先分配大块内存，在上面构造对象
2. 嵌入式系统：在特定硬件地址构造对象
3. 容器实现：如vector在已分配内存上构造元素
4. 游戏引擎：对象池管理游戏对象

注意：
- 不会调用malloc分配内存
- 需要手动调用析构函数：obj->~Type()
- 配合operator new使用实现自定义内存管理

💻 代码示例

// 内存池示例 class ObjectPool { char* memory; size_t objSize; size_t count; public: ObjectPool(size_t n) : objSize(sizeof(T)), count(n) { memory = new char[objSize * n]; // 预分配内存 } template<typename T, typename... Args> T* allocate(Args&&... args) { // 在预分配内存上构造对象 return new (memory + offset) T(std::forward<Args>(args)...); } template<typename T> void deallocate(T* obj) { obj->~T(); // 手动调用析构函数 } }; // 使用 ObjectPool pool(100); MyClass* obj = pool.allocate<MyClass>(args); pool.deallocate(obj);
