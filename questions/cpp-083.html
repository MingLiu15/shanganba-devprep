C++11 模板与SFINAE

C/C++ 困难
标签：C++11, 模板, SFINAE, enable_if, 类型萃取, 元编程

📑 题目

什么是SFINAE？enable_if和类型萃取如何工作？

💡 参考答案

**SFINAE（Substitution Failure Is Not An Error）：**
- 模板参数替换失败不是编译错误
- 只是从重载候选集中移除该模板
- 用于编译期条件选择和类型检查

**std::enable_if：**
- 条件为true时产生有效类型
- 条件为false时产生替换失败（应用SFINAE）
- 用于函数重载的条件选择

**类型萃取（Type Traits）：**
- 编译期获取类型信息
- std::is_integral、std::is_pointer等
- 用于模板元编程和类型检查

**C++20 Concepts：**
- 更清晰的模板约束语法
- 替代复杂的SFINAE技巧

💻 代码示例

#include <iostream> #include <type_traits> #include <vector> // enable_if基本用法 template<typename T> typename std::enable_if<std::is_integral<T>::value, T>::type foo(T t) { std::cout << "Integral version: " << t << std::endl; return t; } template<typename T> typename std::enable_if<!std::is_integral<T>::value, T>::type foo(T t) { std::cout << "Non-integral version: " << t << std::endl; return t; } // 类型萃取示例 template<typename T> void check_type() { std::cout << "is_integral: " << std::is_integral<T>::value << std::endl; std::cout << "is_pointer: " << std::is_pointer<T>::value << std::endl; std::cout << "is_reference: " << std::is_reference<T>::value << std::endl; } // 编译期if（C++17 constexpr if） template<typename T> auto get_value(T t) { if constexpr (std::is_pointer_v<T>) { return *t; // 解引用 } else { return t; // 直接返回 } } // 自定义类型萃取 template<typename T> struct is_container : std::false_type {}; template<typename T, typename Alloc> struct is_container<std::vector<T, Alloc>> : std::true_type {}; // 概念（C++20） // template<typename T> // concept Integral = std::is_integral_v<T>; // // template<Integral T> // T add(T a, T b) { return a + b; } int main() { // enable_if函数重载 foo(42); // 调用integral版本 foo(3.14); // 调用non-integral版本 // 类型萃取 check_type<int>(); check_type<double*>(); // 编译期if int x = 42; std::cout << get_value(x) << std::endl; // 42 std::cout << get_value(&x) << std::endl; // 42（解引用） // 自定义类型萃取 std::cout << "is_container<vector>: " << is_container<std::vector<int>>::value << std::endl; std::cout << "is_container<int>: " << is_container<int>::value << std::endl; return 0; }
