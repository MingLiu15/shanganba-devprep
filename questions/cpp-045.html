函数指针与回调函数

C/C++ 中等
标签：函数指针, 回调, std::function, 函数对象

📑 题目

解释函数指针的概念，以及C++中回调函数的实现方式。

💡 参考答案

函数指针：指向函数的指针，可以像普通变量一样传递和调用。

语法：
- 声明：返回类型 (*指针名)(参数列表)
- 赋值：指针名 = 函数名
- 调用：指针名(参数) 或 (*指针名)(参数)

回调函数：将函数作为参数传递给另一个函数，在特定事件发生时被调用。

C++中回调的多种方式：
1. 函数指针：C风格，简单直接
2. 函数对象（仿函数）：重载operator()的类
3. lambda表达式：C++11，最常用
4. std::function：通用可调用对象包装器
5. 模板：编译期多态

std::function可以包装所有可调用对象（函数指针、仿函数、lambda、成员函数）。

💻 代码示例

// 函数指针 typedef int (*CompareFunc)(int, int); int ascending(int a, int b) { return a - b; } void sort(int* arr, int n, CompareFunc cmp) { // 使用cmp比较元素 for (int i = 0; i < n-1; i++) for (int j = 0; j < n-1-i; j++) if (cmp(arr[j], arr[j+1]) > 0) std::swap(arr[j], arr[j+1]); } // std::function（推荐） #include <functional> std::function<int(int, int)> cmp; cmp = ascending; // 函数指针 cmp = [](int a, int b) { return a - b; }; // lambda // 回调示例 void registerCallback(std::function<void(int)> cb) { cb(42); // 触发回调 }
