Python的import机制

Python 中等
标签：import, 模块导入, sys.path, 循环导入, 导入机制

📑 题目

解释Python的模块导入机制及循环导入问题。

💡 参考答案

Python导入机制：

1. 查找模块：
- 按sys.path顺序查找
- 当前目录 -> PYTHONPATH -> 标准库 -> 第三方包

2. 加载模块：
- 找到模块文件（.py/.pyc/.so/包目录）
- 编译为字节码（如有必要）
- 执行模块顶层代码
- 将模块对象存入sys.modules缓存

3. 创建命名空间：
- 在import语句所在命名空间创建引用

循环导入问题：
- A导入B，B又导入A
- 导致部分模块未完全初始化

解决方案：
1. 重构代码，消除循环依赖
2. 延迟导入（在函数内部导入）
3. 使用importlib动态导入

💻 代码示例
