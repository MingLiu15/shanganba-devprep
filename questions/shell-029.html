Linux基础命令：权限与磁盘管理

Shell脚本 简单
标签：chmod, chown, df, du, 权限管理, 磁盘管理

📑 题目

Linux文件权限如何管理？如何查看磁盘使用情况？

💡 参考答案

**文件权限：**
- rwx：读(4)、写(2)、执行(1)
- 三类用户：所有者(u)、组(g)、其他(o)
- chmod：修改权限
- 数字模式：chmod 755 file
- 符号模式：chmod u+x file
- chown：修改所有者
- chown user:group file

**特殊权限：**
- SUID(4)：以文件所有者权限执行
- SGID(2)：以文件所属组权限执行
- Sticky(1)：目录下文件只能由所有者删除

**磁盘管理：**
- df -h：查看文件系统磁盘空间
- du -sh dir：查看目录大小
- du -h --max-depth=1：查看一级子目录大小

💻 代码示例

# Shell命令示例 commands = """ # 修改权限 chmod 755 script.sh chmod u+x script.sh chmod -R 644 dir/ # 修改所有者 chown user:group file.txt chown -R www-data:www-data /var/www # 查看磁盘 df -h du -sh /home/user du -h --max-depth=1 /var/log | sort -hr | head -10 # 特殊权限 chmod 4755 script # SUID chmod 2755 dir # SGID chmod 1777 /tmp # Sticky """ print(commands)
