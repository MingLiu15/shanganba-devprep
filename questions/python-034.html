collections模块常用类

Python 中等
标签：collections, Counter, defaultdict, deque, OrderedDict

📑 题目

介绍collections模块中的常用类及其使用场景。

💡 参考答案

collections模块提供了Python内置容器的替代方案，功能更强大。

常用类：

1. Counter：计数器
- 用于统计元素出现次数
- 支持数学运算（+、-、&、|）

2. defaultdict：带默认值的字典
- 访问不存在的键时返回默认值
- 避免KeyError

3. deque：双端队列
- 两端插入删除都是O(1)
- 适合实现队列、栈、滑动窗口

4. OrderedDict：有序字典（Python 3.7+后dict本身有序，但仍有用）
- 支持move_to_end()
- 支持popitem(last=False)

5. namedtuple：命名元组
- 为元组元素命名，提高可读性
- 轻量级，不可变

💻 代码示例

from collections import Counter, defaultdict, deque, OrderedDict, namedtuple # Counter - 计数器 text = "hello world" counter = Counter(text) print(counter) # Counter({'l': 3, 'o': 2, 'h': 1, 'e': 1, ' ': 1, 'w': 1, 'r': 1, 'd': 1}) print(counter.most_common(2)) # [('l', 3), ('o', 2)] # defaultdict - 带默认值的字典 dd = defaultdict(list) for key, value in [('a', 1), ('b', 2), ('a', 3)]: dd[key].append(value) print(dd) # defaultdict(<class 'list'>, {'a': [1, 3], 'b': [2]}) # deque - 双端队列 d = deque(maxlen=3) d.append(1) d.append(2) d.appendleft(0) # 左侧添加 print(d) # deque([0, 1, 2], maxlen=3) d.append(3) # 超出maxlen，左侧自动弹出 print(d) # deque([1, 2, 3], maxlen=3) # namedtuple - 命名元组 Point = namedtuple('Point', ['x', 'y']) p = Point(1, 2) print(p.x, p.y) # 1 2 print(p[0], p[1]) # 也可以用索引
