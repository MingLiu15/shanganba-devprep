C++11/14/17新特性

C/C++ 中等
标签：C++11, C++14, C++17, 新特性, 现代C++

📑 题目

列举C++11/14/17的重要新特性。

💡 参考答案

C++11核心特性：
- auto/decltype：自动类型推导
- nullptr：替代NULL
- range-based for：范围for循环
- lambda表达式
- 智能指针：unique_ptr、shared_ptr、weak_ptr
- 右值引用与移动语义
- 可变参数模板
- constexpr：编译期常量
- enum class：强类型枚举
- 初始化列表：统一初始化
- 静态断言：static_assert

C++14特性：
- 泛型lambda：auto参数
- 返回类型推导
- make_unique

C++17特性：
- std::optional：可选值
- std::variant：类型安全联合体
- std::any：任意类型
- 结构化绑定：auto [a, b] = pair
- std::filesystem：文件系统操作
- if constexpr：编译期if
- std::string_view：字符串视图（零拷贝）

💻 代码示例

// C++11 auto x = 42; // auto推导 nullptr_t p = nullptr; // nullptr for (auto& item : vec) { } // 范围for auto func = [](int a, int b) { return a + b; }; constexpr int val = 10; // 编译期常量 // C++14 auto add = [](auto a, auto b) { return a + b; }; auto make() { return 42; } // 返回类型推导 // C++17 std::optional<int> opt = getValue(); if (opt.has_value()) { use(*opt); } std::variant<int, string> v = 42; std::visit([](auto& arg) { /*...*/ }, v); auto [key, value] = std::pair{1, "hello"}; std::string_view sv = "hello"; // 零拷贝
