Linux性能分析工具：perf/strace/vmstat

Shell脚本 困难
标签：性能分析, perf, strace, vmstat, 系统监控, 性能优化

📑 题目

Linux有哪些性能分析工具？如何使用perf、strace和vmstat？

💡 参考答案

**time命令：**
- real：实际经过时间
- user：用户态CPU时间
- sys：内核态CPU时间

**strace：**
- 跟踪系统调用和信号
- 用于诊断程序问题
- `-c`统计系统调用次数
- `-e`过滤特定系统调用

**perf：**
- Linux性能分析利器
- CPU性能计数器
- 火焰图生成
- 函数级性能分析

**vmstat：**
- 系统整体性能监控
- CPU、内存、IO、进程统计
- 采样模式持续监控

💻 代码示例

# 性能分析工具示例 print(''' # ===== time命令 ===== time ./script.sh time -v ./script.sh # ===== strace ===== strace ./program strace -c ./program strace -e open,read,write ./program strace -f ./program strace -o trace.log ./program # ===== perf ===== perf record ./program perf report perf top perf stat -e cache-misses,cache-references ./program # ===== vmstat ===== vmstat vmstat 1 10 vmstat -a # 字段解释： # r: 运行队列长度 # b: 阻塞进程数 # us/sy/id/wa: CPU用户/系统/空闲/等待时间 ''')
