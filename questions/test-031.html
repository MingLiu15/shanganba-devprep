UI自动化测试：元素定位与等待策略

自动化测试 中等
标签：Selenium, 元素定位, 等待策略, Page Object, UI自动化

📑 题目

Selenium中如何有效定位元素？不同等待策略的优缺点？

💡 参考答案

**元素定位策略（按优先级）：**

1. **ID > Name > CSS > XPath**
2. 优先使用稳定的唯一属性
3. 避免使用索引和绝对XPath
4. 使用data-testid或语义化属性

**定位方法：**
- driver.find_element(By.ID, "id")
- By.NAME, By.CLASS_NAME
- By.CSS_SELECTOR
- By.XPATH（相对优于绝对）
- By.LINK_TEXT, By.PARTIAL_LINK_TEXT

**等待策略：**

1. **隐式等待（Implicit）：**
- 全局设置，一次生效
- 简单但可能导致不必要的等待

2. **显式等待（Explicit）：**
- 推荐使用
- 等待特定条件
- ExpectedConditions

3. **流畅等待（Fluent）：**
- 显式等待的增强版
- 自定义轮询间隔和忽略异常

💻 代码示例

from selenium import webdriver from selenium.webdriver.common.by import By from selenium.webdriver.support.ui import WebDriverWait from selenium.webdriver.support import expected_conditions as EC from selenium.webdriver.support.ui import Select from selenium.webdriver.common.action_chains import ActionChains # ===== 元素定位 ===== # 推荐的定位方式（按优先级） def find_elements(driver): # 1. ID（最快最稳定） element = driver.find_element(By.ID, "username") # 2. Name属性 element = driver.find_element(By.NAME, "email") # 3. CSS Selector（灵活且快） element = driver.find_element(By.CSS_SELECTOR, ".btn.primary") element = driver.find_element(By.CSS_SELECTOR, "div#header > ul.nav > li") # 4. XPath（功能强但慢） # 相对XPath（推荐） element = driver.find_element(By.XPATH, "//button[contains(text(), 'Submit')]") element = driver.find_element(By.XPATH, "//div[@class='card'][2]/button") # 避免：绝对XPath # element = driver.find_element(By.XPATH, "/html/body/div[1]/div[2]/form/button") # ===== 等待策略 ===== # 隐式等待（全局设置） def test_implicit_wait(driver): driver.implicitly_wait(10) # 全局等待10秒 driver.get("https://example.com") # 显式等待（推荐） def test_explicit_wait(driver): driver.get("https://example.com") # 等待元素可见 element = WebDriverWait(driver, 10).until( EC.visibility_of_element_located((By.ID, "username")) ) # 等待元素可点击 button = WebDriverWait(driver, 10).until( EC.element_to_be_clickable((By.CSS_SELECTOR, ".btn-submit")) ) # 等待文本出现 WebDriverWait(driver, 10).until( EC.text_to_be_present_in_element((By.ID, "status"), "Success") ) # 等待元素消失 WebDriverWait(driver, 10).until( EC.invisibility_of_element_located((By.CLASS_NAME, "loading")) ) # ===== Page Object模式 ===== class LoginPage: def __init__(self, driver): self.driver = driver self.wait = WebDriverWait(driver, 10) @property def username(self): return self.wait.until( EC.visibility_of_element_located((By.ID, "username")) ) @property def password(self): return self.wait.until( EC.visibility_of_element_located((By.ID, "password")) ) @property def submit_btn(self): return self.wait.until( EC.element_to_be_clickable((By.CSS_SELECTOR, ".btn-login")) ) def login(self, username, password): self.username.send_keys(username) self.password.send_keys(password) self.submit_btn.click() from pages.DashboardPage import DashboardPage return DashboardPage(self.driver)
