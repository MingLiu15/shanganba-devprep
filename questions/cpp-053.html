STL算法函数

C/C++ 中等
标签：STL算法, algorithm, find, sort, accumulate

📑 题目

介绍常用的STL算法函数。

💡 参考答案

STL <algorithm>头文件提供大量通用算法：

查找算法：
- find/find_if：查找元素
- binary_search：二分查找（需有序）
- lower_bound/upper_bound：查找插入位置

排序算法：
- sort：快速排序
- stable_sort：稳定排序
- partial_sort：部分排序

数值算法（<numeric>）：
- accumulate：求和/累积
- inner_product：内积
- partial_sum：前缀和

修改算法：
- copy/copy_if：复制
- transform：转换
- replace/replace_if：替换
- remove/remove_if：删除（需配合erase）

集合算法：
- set_union：并集
- set_intersection：交集
- set_difference：差集

其他：
- reverse：反转
- unique：去重（需配合erase）
- next_permutation：下一个排列

💻 代码示例

#include <algorithm> #include <numeric> #include <vector> std::vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6}; // 查找 auto it = std::find(v.begin(), v.end(), 5); auto it2 = std::find_if(v.begin(), v.end(), [](int x){ return x > 5; }); // 排序 std::sort(v.begin(), v.end()); std::sort(v.begin(), v.end(), std::greater<int>()); // 二分查找（需先排序） bool exists = std::binary_search(v.begin(), v.end(), 5); // 数值算法 int sum = std::accumulate(v.begin(), v.end(), 0); int product = std::accumulate(v.begin(), v.end(), 1, std::multiplies<int>()); // 转换 std::vector<int> squares; std::transform(v.begin(), v.end(), std::back_inserter(squares), [](int x) { return x * x; }); // 删除-擦除惯用法 v.erase(std::remove(v.begin(), v.end(), 1), v.end()); // 去重 v.erase(std::unique(v.begin(), v.end()), v.end());
