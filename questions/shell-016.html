函数定义与参数传递

Shell脚本 简单
标签：函数, 参数传递, 返回值, 局部变量

📑 题目

Shell中如何定义函数？参数如何传递？

💡 参考答案

**定义函数：**
```bash
function name() {
commands
}
# 或
name() {
commands
}
```

**参数传递：**
- 函数内使用 `$1, $2, ...` 获取参数
- `$@` 所有参数
- `$#` 参数个数

**返回值：**
- `return n` - 返回退出状态码（0-255）
- 使用 `echo` 返回字符串结果
- 捕获结果：`result=$(func)`

**局部变量：**
- `local var=value` - 定义局部变量
- 避免污染全局命名空间

**递归：**
- Shell支持函数递归调用
