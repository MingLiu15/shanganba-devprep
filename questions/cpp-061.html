KMP字符串匹配算法

C/C++ 困难
标签：KMP, 字符串匹配, 算法, next数组

📑 题目

解释KMP算法的原理及其next数组的构建过程。

💡 参考答案

KMP算法：一种高效的字符串匹配算法，时间复杂度O(m+n)。

核心思想：
- 利用已匹配的信息，避免主串指针回溯
- 当匹配失败时，利用next数组确定模式串应该从哪个位置继续匹配

next数组（前缀函数）：
- next[i]表示模式串[0..i]的最长相等前后缀长度
- 即最长的k，使得pattern[0..k-1] == pattern[i-k+1..i]

构建next数组：
- 双指针法，i遍历模式串，j指向前缀末尾
- 匹配时i和j同时前进
- 不匹配时j回退到next[j-1]

匹配过程：
- 主串指针i不回溯，只移动模式串指针j
- 匹配失败时j = next[j]，继续匹配

时间复杂度：
- 构建next数组：O(m)
- 匹配过程：O(n)
- 总计O(m+n)

💻 代码示例

// 构建next数组 vector<int> buildNext(const string& pattern) { int m = pattern.size(); vector<int> next(m, 0); int j = 0; for (int i = 1; i < m; i++) { while (j > 0 && pattern[i] != pattern[j]) j = next[j - 1]; // 回退 if (pattern[i] == pattern[j]) j++; next[i] = j; } return next; } // KMP匹配 int kmpSearch(const string& text, const string& pattern) { vector<int> next = buildNext(pattern); int j = 0; // 模式串指针 for (int i = 0; i < text.size(); i++) { while (j > 0 && text[i] != pattern[j]) j = next[j - 1]; // 模式串回退，主串不回退 if (text[i] == pattern[j]) j++; if (j == pattern.size()) return i - j + 1; // 找到匹配 } return -1; // 未找到 }
