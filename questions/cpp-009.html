智能指针的种类和区别

C/C++ 中等
标签：智能指针, unique_ptr, shared_ptr, weak_ptr, 内存管理

📑 题目

说明C++11三种智能指针的特点和使用场景。

💡 参考答案

1. unique_ptr：独占所有权，不可复制只可移动，开销最小，默认首选
2. shared_ptr：共享所有权，引用计数，可复制，线程安全（计数器原子操作），有循环引用问题
3. weak_ptr：弱引用，不增加引用计数，用于打破循环引用，需要时lock()转为shared_ptr

使用建议：
- 默认用unique_ptr
- 需要共享时用shared_ptr
- 避免循环引用用weak_ptr

💻 代码示例

unique_ptr<int> up(new int(10)); // unique_ptr<int> up2 = up; // 错误，不可复制 unique_ptr<int> up2 = move(up); // 可以移动 shared_ptr<int> sp1 = make_shared<int>(20); shared_ptr<int> sp2 = sp1; // 引用计数+1 struct Node { shared_ptr<Node> next; weak_ptr<Node> parent; // 避免循环引用 };
