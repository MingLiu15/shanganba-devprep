Python包管理与虚拟环境

Python 简单
标签：工具, 包管理, pip, 虚拟环境, venv

📑 题目

介绍Python包管理工具pip和虚拟环境的使用方法。

💡 参考答案

pip是Python的包管理工具，用于安装、卸载和管理第三方包。

常用pip命令：
- pip install package：安装包。
- pip install -r requirements.txt：从文件批量安装。
- pip uninstall package：卸载包。
- pip list：列出已安装的包。
- pip freeze > requirements.txt：导出依赖。
- pip show package：查看包详情。

虚拟环境：
- 为每个项目创建独立的Python环境，避免依赖冲突。
- venv（Python内置）：python -m venv myenv。
- virtualenv（第三方）：功能更丰富。
- conda：Anaconda/Miniconda提供，适合数据科学。

虚拟环境使用：
1. 创建：python -m venv myenv。
2. 激活：source myenv/bin/activate（Linux/Mac）或 myenv\Scripts\activate（Windows）。
3. 安装包：pip install package。
4. 导出依赖：pip freeze > requirements.txt。
5. 退出：deactivate。

依赖管理工具：
- poetry：现代Python项目管理工具，集成了依赖管理和打包。
- pipenv：pip和virtualenv的结合。
- pdm：基于PEP 582的包管理器。

pyproject.toml是现代Python项目的标准配置文件（PEP 518/621）。
