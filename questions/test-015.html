📑 题目

什么是场景法？如何设计场景测试用例？

💡 参考答案

**场景法定义：**
根据用户实际使用场景来设计测试用例，模拟用户操作流程，验证业务逻辑。

**场景类型：**
1. **基本流（主场景）**：正常流程，用户按预期操作
2. **备选流（备选场景）**：异常或分支流程
3. **异常流**：错误处理流程

**设计步骤：**
1. 分析业务流程，绘制流程图
2. 确定基本流
3. 识别备选流和异常流
4. 组合场景生成测试用例

**示例：用户登录场景**
- 基本流：输入正确账号密码 → 登录成功
- 备选流：账号不存在、密码错误、账号锁定
- 异常流：网络超时、服务不可用

# 场景法测试示例 import pytest class UserService: def __init__(self): self.users = {"admin": "password123"} self.locked_users = set() self.login_attempts = {} def login(self, username, password): # 检查账号是否锁定 if username in self.locked_users: raise ValueError("账号已锁定") # 检查账号是否存在 if username not in self.users: raise ValueError("账号不存在") # 检查密码 if self.users[username] != password: # 记录失败次数 self.login_attempts[username] = self.login_attempts.get(username, 0) + 1 if self.login_attempts[username] >= 3: self.locked_users.add(username) raise ValueError("密码错误次数过多，账号已锁定") raise ValueError("密码错误") # 登录成功，重置失败次数 self.login_attempts[username] = 0 return {"status": "success", "user": username} @pytest.fixture def user_service(): return UserService() class TestLoginScenarios: """登录场景测试""" def test_basic_flow_success(self, user_service): """基本流：正常登录成功""" result = user_service.login("admin", "password123") assert result["status"] == "success" def test_alternate_user_not_exist(self, user_service): """备选流：账号不存在""" with pytest.raises(ValueError, match="账号不存在"): user_service.login("unknown", "password") def test_alternate_wrong_password(self, user_service): """备选流：密码错误""" with pytest.raises(ValueError, match="密码错误"): user_service.login("admin", "wrongpassword") def test_alternate_account_locked(self, user_service): """备选流：账号锁定（连续3次密码错误）""" # 连续输入错误密码3次 for _ in range(3): try: user_service.login("admin", "wrong") except ValueError: pass # 第4次尝试登录 with pytest.raises(ValueError, match="账号已锁定"): user_service.login("admin", "password123") # 组合场景测试 @pytest.mark.parametrize("username,password,expected", [ ("admin", "password123", "success"), # 正常登录 ("unknown", "password", "账号不存在"), # 账号不存在 ("admin", "wrong", "密码错误"), # 密码错误 ]) def test_login_scenarios(self, user_service, username, password, expected): if expected == "success": result = user_service.login(username, password) assert result["status"] == "success" else: with pytest.raises(ValueError, match=expected): user_service.login(username, password)
