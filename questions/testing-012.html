📑 题目

常见的安全测试类型和工具有哪些？

💡 参考答案

常见安全漏洞（OWASP Top 10）：
1. 注入攻击：SQL、NoSQL、OS命令注入
2. 失效的身份认证：弱密码、会话管理
3. 敏感数据泄露：明文传输、弱加密
4. XML外部实体（XXE）
5. 失效的访问控制：越权访问
6. 安全配置错误
7. 跨站脚本（XSS）
8. 不安全的反序列化
9. 使用含有已知漏洞的组件
10. 不足的日志记录和监控

安全测试类型：
1. 静态分析（SAST）：扫描源代码
2. 动态分析（DAST）：运行时扫描
3. 依赖扫描：检查第三方库漏洞
4. 渗透测试：模拟攻击

💻 代码示例

import requests # SQL注入测试 def test_sql_injection(): payloads = ["' OR '1'='1", "' OR 1=1--", "1; DROP TABLE users--"] for payload in payloads: response = requests.post('/login', data={'username': payload, 'password': 'test'}) assert 'error' in response.json() or response.status_code == 400 # XSS测试 def test_xss_protection(): xss_payload = '<script>alert("xss")</script>' response = requests.post('/comment', data={'text': xss_payload}) assert '<script>' not in response.text or '&lt;script&gt;' in response.text # 认证测试 def test_authentication(): weak_passwords = ['123456', 'password', 'admin'] for pwd in weak_passwords: response = requests.post('/register', json={'password': pwd}) assert response.status_code == 400 # 授权测试 def test_authorization(): user_a_token = login('user_a', 'pass') response = requests.get('/users/user_b/private', headers={'Authorization': user_a_token}) assert response.status_code == 403
