xargs高级用法与expect自动化交互

Shell脚本 困难
标签：xargs, expect, 自动化交互, 并行处理, 高级用法

📑 题目

xargs的高级用法有哪些？expect如何实现自动化交互？

💡 参考答案

**xargs高级用法：**

1. **并行处理：**
- -P n：并行执行n个进程
- 大幅提升批量处理速度

2. **占位符控制：**
- -I {}：自定义占位符
- 支持多次引用

3. **空字符分隔：**
- -0：处理含空格的文件名
- 配合find -print0使用

4. **批量参数：**
- -n 1：每次传递1个参数
- -L 1：每次传递1行

**expect自动化交互：**
- 自动响应交互式程序
- spawn：启动程序
- expect：等待特定输出
- send：发送输入
- 适用于ssh、ftp、passwd等

💻 代码示例

# xargs高级用法示例 xargs_examples = """ # 并行处理（4个并发） find . -name "*.log" | xargs -P 4 gzip # 自定义占位符 cat files.txt | xargs -I {} cp {} /backup/{} # 处理含空格的文件名 find . -name "*.txt" -print0 | xargs -0 rm # 每次处理1个文件 ls *.py | xargs -n 1 python # 结合grep递归搜索 find . -type f | xargs grep -l "pattern" # 批量重命名 ls | xargs -I {} mv {} prefix_{} """ # expect脚本示例 expect_examples = """ #!/usr/bin/expect # SSH自动登录 spawn ssh user@remote_host expect "password:" send "mypassword\r" expect "$ " send "ls -la\r" expect "$ " interact # 自动FTP上传 spawn ftp ftp.example.com expect "Name" send "username\r" expect "Password:" send "password\r" expect "ftp>" send "put localfile remotefile\r" expect "ftp>" send "bye\r" # 自动修改密码 spawn passwd expect "New password:" send "newpass123\r" expect "Retype new password:" send "newpass123\r" expect eof """ print(xargs_examples) print(expect_examples)
